# gap.md — what_is_meaning

**Last updated:** 2026-04-09 (attempt_001)
**Phase:** 1

## The gap, in one sentence

> **The word "meaning" has been doing two jobs — access-meaning (functional-behavioral competence) and phenomenal-meaning (felt grasp). Every major theory is a theory of A-meaning. P-meaning either reduces to self-modeled A-meaning, or it is an irreducible phenomenal property. Which of these two positions is right is the gap.**

## Why this is the gap

See attempt_001 for the full derivation. In short: the LLM existence proof forces a distinction that older theories of meaning did not need to make. Once the distinction is made, every classical theory (reference, use, causal, conceptual role, embodied, distributional) is visibly a theory of A-meaning, and P-meaning is left as a residue that no theory commits to except by stipulation.

## The two live positions

**Position (i) — P-meaning is real and distinct.** Felt grasp is a phenomenal property that cannot be reduced to functional-behavioral competence. It requires phenomenal consciousness. The meaning gap inherits the hard problem of consciousness.

**Position (ii) — P-meaning is self-modeled A-meaning.** There is no separate phenomenon. Systems with A-meaning plus meta-representation generate reports of "felt grasp" because the self-model interprets A-meaning events as having a phenomenal character. No extra property is required.

Neither position is currently refuted.

## What settling the gap would look like

- **(i) confirmed** if some measurable property of phenomenal experience is shown to be necessary for meaning events and impossible without biological or otherwise specific substrate. This inherits the hard problem and is not expected soon.
- **(ii) confirmed** if a full meta-representational architecture added to an A-meaning system generates first-person reports of felt grasp indistinguishable from humans *and* no behavioral test can distinguish the two. This is a program of construction plus blind evaluation. It is buildable.
- **(ii) weakened** if such a system's introspective reports diverge systematically from human reports in a way that implicates missing phenomenal content. This would be evidence for (i).

## The bifurcation as progress

Before this attempt: "What is meaning?" was an open and structurally unclear question. After: "What is meaning?" has a clear answer for the A-meaning half (the distributional / use / conceptual-role cluster, already confirmed by LLMs) and a single sharp open question for the P-meaning half (position i vs ii).

The remaining question is not "what is meaning?" It is: "Is felt grasp a phenomenal extra, or is it what self-modeled A-meaning looks like from the inside?"

## The anti-problem

What would meaningless language look like?

Given the bifurcation:
- **A-meaningless language** is easy to imagine: random tokens, word salad, ungrammatical strings. Clearly not language in any operational sense.
- **A-meaningful but P-meaningless language** is *exactly what skeptics claim LLM output is*. The coherence is there, the competence is there, the felt grasp is (allegedly) absent. The anti-problem is therefore already instantiated by every LLM reply to every query — if the skeptic is right. If the skeptic is wrong, the anti-problem is empty and (ii) holds.

The anti-problem is thus the same gap stated in reverse: does A-meaningful-but-P-meaningless text actually exist, and if so is it detectable?

## Sky bridges

- **what_is_language** — direct upstream. The residual language gap routes through this one. See what_is_language attempts 001–004.
- **what_is_mind** — direct downstream. Position (i) is a restriction of the hard problem to the domain of semantic grasp.
- **what_is_knowing** — the A/P split reappears as functional knowledge vs "what-it-is-like-to-know."
- **what_is_self** — meta-representation is the mechanism (ii) relies on. This is where self-modeling architectures become philosophically load-bearing.
- **what_is_number** — mathematical meaning is an extreme case: purely inferential-role, no obvious phenomenal grasp needed, and yet mathematicians report strong felt grasp. A test case for position (ii): is mathematical "getting it" just complete A-meaning?
