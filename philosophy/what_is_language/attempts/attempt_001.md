# attempt_001 — The LLM Existence Proof

**Date:** 2026-04-09
**Status:** Foundation. Establishes the empirical claim every theory of language must now account for.

## The claim

Large language models — transformer architectures trained by next-token prediction over static text corpora — learn to produce grammatical, semantically coherent, contextually appropriate language. This is not a thought experiment. Every deployed LLM is an existence proof. Treat it as raw data that any theory of language must now absorb.

## The minimal training conditions

An LLM learns language from:

- **Text only.** No audio, no video, no touch, no embodiment, no sensorimotor loop.
- **No feedback loop with a physical world.** Actions have no consequences during pretraining.
- **No online social interaction.** The corpus is static. No interlocutors, no joint attention, no gaze following, no corrective feedback from caregivers during learning.
- **No explicit grammatical supervision.** No parse trees, no POS tags as primary signal, no rule tables. The loss function is next-token cross-entropy on raw text.
- **No hand-crafted grammar module.** The transformer architecture was designed for general sequence modelling. It was not designed around linguistic universals, syntactic rules, or a language-acquisition device.

From these conditions alone, fluent language emerges.

## What the existence proof establishes

**Undisputed empirical content.** It is possible to reach fluent language *production* via pure distributional learning over a static text corpus, without grounding, without sociality, without innate grammar, and without explicit rules.

**Disputed interpretation.** Whether what LLMs have counts as "having language" in the sense that matters for the philosophical question. This is precisely where every competing ontology places its stake.

## Bearing on each competing ontology

### Strong Universal Grammar (Chomsky, poverty-of-stimulus line)

**Claim attacked:** "No finite learning system without innate grammatical structure can acquire grammar from data alone, because the data is too sparse to uniquely determine the target."

**Status:** Weakened, not killed. LLMs do it — but they see orders of magnitude more data than any child. The poverty-of-stimulus argument is about *human* data volumes. LLMs evade the argument by scaling past it. See attempt_002 for the numbers.

**What survives:** the Chomskyan can retreat to "UG is what makes human learning sample-efficient; LLMs are a different beast using brute data."

### Strong embodiment / grounded cognition (Lakoff, Barsalou, Glenberg)

**Claim attacked:** "Linguistic competence requires sensorimotor grounding. Meaning is metaphor grounded in bodily experience."

**Status:** Weakened. LLMs have no body and produce coherent text that respects commonsense physics, spatial reasoning, and metaphorical extension.

**What survives:** the embodiment theorist can retreat to "production is not understanding; without grounding there is no reference, only syntax." This is a meaning claim, not a behavior claim. It cannot be falsified by behavioral evidence alone. See attempt_003 for whether this retreat is stable.

### Usage-based / social-pragmatic (Tomasello)

**Claim attacked:** "Language requires joint attention, shared intentionality, and dyadic social interaction to be acquired."

**Status:** Weakened. LLMs have none of these during training.

**What survives:** "LLMs train on text generated by social creatures, so sociality enters the pipeline laundered through the corpus." This is a real point — the corpus is not socially neutral — but it admits that the sociality can be *encoded and learned from text*, which is itself a major concession.

### Distributional hypothesis (Firth, Harris, Sahlgren, Lenci)

**Claim strengthened:** "You shall know a word by the company it keeps."

**Status:** Strengthened by the strongest possible existence proof. LLMs are the maximal experimental test of whether distributional regularities in text suffice for language. They succeed in producing language. This does not resolve whether distribution suffices for *meaning* — only that it suffices for the production of language-shaped behavior.

### Computational / rule-based (classical symbolic)

**Claim reframed:** "Language is governed by rules."

**Status:** LLMs do not use explicit rules, but their weights plausibly implement rules implicitly. The debate shifts from "are there rules" to "what does it take for a system to *have* a rule." Dennett's instrumentalism vs Fodor's realism, replayed at the weight level.

### Social convention (Lewis, Millikan)

**Claim compatible.** Language is shared practice. LLMs participate in the practice by being trained on its traces and then re-entering it through use. The LLM existence proof does not strain this view and may modestly support it.

## What this attempt does NOT settle

The existence proof is about **language production as observable behavior**. It does not and cannot show:

1. That LLMs *understand* what they produce.
2. That LLM "language" is the same kind as human language.
3. That meaning reduces to distribution.
4. That grounding is unnecessary for reference.
5. That consciousness is irrelevant to language.
6. That there is no functional difference between human and LLM language use.

Each of these is an interior claim. Behavioral existence proofs only reach the behavior.

## The gap this sharpens

Before LLMs, the central question was "can language be learned without innate grammar / grounding / sociality?" That question now has a partial answer: **yes, something language-shaped can.** The remaining question is:

> **What, if anything, distinguishes LLM-generated language from human-generated language, and can that distinction be expressed as a measurable quantity?**

This is the gap. attempt_002 proposes the sharpest current measurement.

## Next attempt

**attempt_002** — quantify the one cleanly measurable difference that is available right now: **sample complexity.** Humans learn fluent language from ~10⁷–10⁸ words. Frontier LLMs use ~10¹²–10¹³. The gap is 4–6 orders of magnitude and it is not closing fast. That number is the first real handle on the wall.
