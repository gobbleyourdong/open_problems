# philosophy/ — Meaning, value, mind, knowing

> The questions that survive every empirical answer.

Philosophy's questions don't get crossed off when science answers an adjacent question. Neuroscience can map the brain but the hard problem of consciousness remains. Game theory can model cooperation but the ground of ethics stays open. These are tier-0 generative questions — they generate disciplines without being contained by any of them.

## The questions

| Question | What's at stake |
|----------|----------------|
| [what_is_mind](./what_is_mind/) | The hard problem of consciousness. Functionalism, IIT, panpsychism, illusionism. |
| [what_is_knowing](./what_is_knowing/) | Epistemology after Gettier. What separates knowledge from true belief? |
| [what_is_meaning](./what_is_meaning/) | Reference, use, distributional semantics. Do LLMs mean what they say? |
| [what_is_good](./what_is_good/) | Moral realism, error theory, Hume's is-ought gap. |
| [what_is_beauty](./what_is_beauty/) | Aesthetics, symmetry, evolutionary signals, cultural construction. |
| [what_is_self](./what_is_self/) | Personal identity, Parfit's bundle, the transporter paradox. |
| [what_is_language](./what_is_language/) | Universal grammar vs distributional. What do LLMs prove about language? |
| [what_is_number](./what_is_number/) | Platonism, formalism, intuitionism, structuralism. Is math discovered or invented? |
| [what_is_life](./what_is_life/) | Demarcation question. Metabolism, replication, autopoiesis, computational substrate. |

## How to work these

Each question directory has:
- `PROBLEM.md` — the question, the ladder from tier-0 down, competing ontologies, the gap, the anti-problem
- `attempts/` — numbered attempts, dead ends, sky bridges

These are **generative questions** — they don't get closed, they get mapped. The contribution is the map with the gaps marked precisely.
