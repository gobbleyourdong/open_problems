# Poincaré Conjecture

## Statement
Every simply connected, closed 3-manifold is homeomorphic to the 3-sphere S³.

## Status: SOLVED (Perelman, 2003) — Clay Prize awarded 2010 (declined)

## Solution
Perelman proved the conjecture via Ricci flow with surgery:
1. Hamilton's Ricci flow: ∂g/∂t = -2 Ric(g)
2. Perelman's entropy functional and κ-noncollapsing
3. Canonical neighborhood theorem
4. Surgery procedure for singularities
5. Long-time behavior: flow converges to constant curvature
