## THEOREM

For any N ≤ 3 divergence-free modes on T³ with unit amplitudes,
at x* = argmax|ω|²:

    S²ê(x*) < (3/4)|ω(x*)|²

## PROOF

**Step 1 (Argmax alignment).** At the vertex max with signs sⱼ = ±1:
flipping any sⱼ cannot increase |ω|². For sⱼ = +1:
|ω-2vⱼ|² ≤ |ω|² gives cosγⱼ ≥ 1/|ω|. For sⱼ = -1:
|ω+2vⱼ|² ≤ |ω|² gives cosγⱼ ≤ -1/|ω|.
In both cases: **|cosγⱼ| ≥ 1/|ω|** and **sinγⱼ ≤ √(1-1/|ω|²)**.

**Step 2 (Self-vanishing + triangle).** From the Biot-Savart Coupling
Lemma: |Sⱼ·ê| = (aⱼ/2)sinγⱼ. By the triangle inequality:

    S²ê ≤ (Σⱼ (1/2)sinγⱼ)² ≤ (N/2)² (1-1/|ω|²)

using sinγⱼ ≤ √(1-1/|ω|²) and Cauchy-Schwarz.

**Step 3 (The bound).** For S²ê < (3/4)|ω|²: need

    N²(|ω|²-1)/(4|ω|²) < (3/4)|ω|²

i.e., **N²(|ω|²-1) < 3|ω|⁴**, i.e., **3t² - N²t + N² > 0** where t = |ω|².

**Step 4 (Discriminant).** The discriminant is N⁴ - 12N² = N²(N²-12).

For **N ≤ 3**: N² ≤ 9 < 12, so **discriminant < 0**. No real roots.
Since 3t² - N²t + N² > 0 at t=0 (value N² > 0): it's positive for all t.

Therefore: S²ê < (3/4)|ω|² for all configurations with N ≤ 3. **∎**

## REMARKS

1. **No case split**: the proof handles ALL sign patterns simultaneously.
   The key: sinγ depends on |cosγ|, not its sign. Anti-aligned modes
   (sⱼ=-1, cosγⱼ<0) satisfy the same sinγ bound as aligned modes.

2. **No k-geometry dependence**: the proof uses ONLY the self-vanishing
   identity (which holds for any k) and the argmax condition.

3. **The bound is tight**: for N=3 at |ω|²→∞: Budget²/(3|ω|²/4) →
   9/(4×3/4×|ω|²) × (|ω|²-1) → 3×1/3 = 1. So the ratio approaches 1
   for large |ω|². The discriminant argument gives 0 margin in the limit.

4. **N=4 is the boundary**: for N=4, discriminant = 16(16-12) = 64 > 0.
   The polynomial 3t²-16t+16 has roots t=4 and t=4/3. At t=|ω|²=4
   (the minimum from |ω|²≥N=4): equality. The proof JUST FAILS for N=4.
