## Step 3: Explicit Hodge Classes (Group Theory + Analysis)

The Griffiths residue representation: H^{p,q}(X) for X a hypersurface in CP^{n+1}
of degree d is isomorphic to a quotient of the Jacobian ring:

  H^{n-p, p}_{prim}(X) ≅ R^{(p+1)d - n - 2} / J

where R = C[x₀,...,x_{n+1}] and J = (∂f/∂x₀, ..., ∂f/∂x_{n+1}).

For the Fermat cubic (d=3, n=4):
  H^{2,2}_{prim} ≅ R^{3·3 - 6} / J = R^3 / J

R^3 = degree-3 monomials in 6 variables. There are C(8,3) = 56 monomials.
J in degree 3: generated by x_i² (the partials of x_i³ are 3x_i²).

R^3 / J = span of degree-3 monomials NOT divisible by any x_i².
These are: monomials x_a x_b x_c with a, b, c DISTINCT.
Count: C(6,3) = 20 monomials.

So h^{2,2}_{prim} = 20. With the non-primitive class h²: h^{2,2} = 21. ✓

The 20 primitive (2,2)-classes correspond to the 20 monomials x_a x_b x_c
(a < b < c, chosen from {0,1,2,3,4,5}).

**These are ALL Hodge classes** (they generate the full H^{2,2} over Q,
since the Fermat's symmetry group acts transitively enough).
