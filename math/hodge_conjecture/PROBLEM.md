# Hodge Conjecture

## Statement
On a non-singular complex projective algebraic variety, every Hodge class is a rational linear combination of classes of algebraic cycles.

## Status: OPEN (Clay Millennium Prize, $1M)

## Key Objects
- X: non-singular complex projective variety
- H^{p,q}(X): Hodge decomposition of cohomology
- Hodge class: element of H^{2p}(X,Q) ∩ H^{p,p}(X)
- Algebraic cycle: formal sum of subvarieties
- Cycle class map: cl: Z^p(X) → H^{2p}(X,Q)

## Known Results
- True for p = 1 (Lefschetz (1,1)-theorem)
- True for p = dim(X) - 1 (by hard Lefschetz)
- False for integer coefficients (Atiyah-Hirzebruch counterexample)
- True for abelian varieties of dimension ≤ 5
- Generalized Hodge conjecture: open
- Grothendieck's standard conjectures would imply Hodge

## Approaches
- Algebraic geometry (intersection theory, motives)
- Hodge theory (variation of Hodge structure, period maps)
- K-theory (algebraic K-theory, Chern character)
- Derived categories
- Motivic cohomology
