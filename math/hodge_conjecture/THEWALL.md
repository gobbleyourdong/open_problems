## The Problem

On a smooth complex projective variety X, every Hodge class is a rational
linear combination of algebraic cycle classes.

### The Cycle Algebra (attempt_010)

Hodge reframed as a WORD PROBLEM:

```
Algebra C(X): generated by subvarieties, closed under ∩, f*, f_*, c_k(E), correspondences
Evaluation: cl : C(X) → H*(X, Q)
Word problem: find w ∈ C(X) with cl(w) = α for each α ∈ Hdg^p(X)
```

Semi-decidable: terminates when spanning set found, runs forever if Hodge fails.
Fully decidable with degree bounds (known for abelian varieties, cubic fourfolds).

### The Tannakian Formulation (attempt_014)

**Hodge ⟺ Hg(X) = G_mot(X)**

The Hodge group equals the motivic Galois group. Equivalently:
the functor Motives → Hodge Structures is fully faithful.

For abelian varieties: reduces to checking MT-invariants are algebraic.
Finite computation per dimension g. PROVED for g ≤ 5.

## The Wall

### For Abelian Varieties (the "soft" wall)

**Weil classes on simple CM abelian varieties of dimension g ≥ 6.**

These are Hodge classes in H^{g,g} that arise from the CM structure.
They are absolute Hodge (Deligne 1982) but algebraicity is open.
The generator can attack them one CM type at a time (finite per g).

This wall is SOFT: it yields to computation, dimension by dimension.
The obstruction is COMPLEXITY (the number of CM types grows with g),
not IMPOSSIBILITY.

### For General Varieties (the "hard" wall)

**The motivic t-structure does not exist yet.**

The general Hodge conjecture requires: for EVERY smooth projective X,
Hg(X) = G_mot(X). The Hodge group can be ANY reductive Q-group.
There's no finite classification to exhaust.

The general proof would require the motivic t-structure on Voevodsky's
derived category of motives — Grothendieck's dream of a universal
cohomology theory. This object has been CONJECTURED since the 1960s
but never constructed.

**The wall IS the motivic t-structure.** Not a single inequality.
Not a single computation. The EXISTENCE of a mathematical structure.

### Why p ≥ 2 Is Hard (the structural obstruction)

| Feature | p = 1 (PROVED) | p ≥ 2 (OPEN) |
|---------|---------------|--------------|
| Cycles ↔ sheaves | Divisors ↔ line bundles | No analog |
| Classification | Picard group (abelian variety) | Chow group (wild) |
| Exponential sequence | H¹(O*) → H²(Z) → H²(O) | Nothing |
| Deformation | Linear systems (move freely) | Rigid (isolated) |
| Abel-Jacobi | Isomorphism onto Jacobian | Not full, not faithful |
| Intermediate Jacobian | Abelian variety (algebraic) | Complex torus (NOT algebraic) |

The p = 1 proof uses the EXPONENTIAL SEQUENCE, which links topology
(H²(Z)) to algebra (Pic(X) = H¹(O*)) via analysis (H²(O)). For p ≥ 2,
NO SUCH SEQUENCE EXISTS. The three worlds (topology, algebra, analysis)
are disconnected in higher codimension.

## The Anti-Problem

"What would a counterexample look like?"

A smooth projective variety X with a class α ∈ H^{2p}(X, Q) ∩ H^{p,p}(X)
that is NOT in the image of the cycle class map cl : CH^p(X) ⊗ Q → H^{2p}.

Properties of the counterexample:
- α is rational and (p,p) — passes all NECESSARY conditions
- α is absolute Hodge (Deligne) — passes the Galois test
- But NO subvariety (or combination) has [Z] = α
- Must have p ≥ 2 (p = 1 is proved)
- Integer Hodge fails (Atiyah-Hirzebruch) but rational Hodge might not

**Where to look**: varieties with large H^{p,p} and few known cycles.
The generator would report: rank(Alg) < rank(Hdg) with the SPECIFIC
missing class identified. Then: prove no cycle can ever realize it.

No such example is known. The conjecture has ZERO failures across
all verified cases (thousands of varieties in LMFDB + explicit checks).
