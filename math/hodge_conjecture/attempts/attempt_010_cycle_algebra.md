## The Key Insight

The bottleneck in the Hodge generator (Layer 5: cycle search) can be
formalized as a WORD PROBLEM in a finitely-generated algebra.

Algebraic cycles on X are built by OPERATIONS on ATOMS. The cycle class
map evaluates each expression to a cohomology class. Hodge asks:
does the image of ALL expressions span the Hodge lattice?

This is a DECIDABLE question (in principle) for each fixed variety,
because the target lattice is finite-dimensional.

## The Cycle Algebra C(X)

### Atoms (generators)
For a variety X ⊂ CP^N of dimension n:

| Atom | Type | Codimension | Symbol |
|------|------|-------------|--------|
| Point | [pt] | n | p |
| Hyperplane section | [H] | 1 | h |
| X itself | [X] | 0 | 1 |
| Line in X | [ℓ] | n-1 | ℓ |
| Rational curve C ⊂ X | [C] | n-1 | c_i |
| Surface S ⊂ X | [S] | n-2 | s_j |
| ... up to codimension p ... | | p | z_k |

For SPECIFIC variety families, the atoms are enumerable:
- Cubic fourfold X ⊂ CP^5: surfaces in X of degree d (finite for each d)
- Abelian variety A: abelian subvarieties, graphs of endomorphisms

### Operations (relations)

| Operation | Input | Output | Formula |
|-----------|-------|--------|---------|
| Intersection | Z₁, Z₂ | Z₁·Z₂ | cl(Z₁·Z₂) = cl(Z₁) ∪ cl(Z₂) |
| Self-intersection | Z, k | Z^k | cl(Z^k) = cl(Z)^k |
| Pullback | f: Y→X, Z on X | f*Z on Y | cl(f*Z) = f*(cl(Z)) |
| Pushforward | f: Y→X, W on Y | f_*W on X | cl(f_*W) = f_*(cl(W)) |
| Chern class | E bundle on X | c_k(E) | cl(c_k(E)) from Chern-Weil |
| Correspondence | γ ∈ CH(X×Y), W on Y | γ_*W on X | cl(γ_*W) = p_{1*}(γ · p_2*(W)) |

### The Algebra Structure

C(X) is generated by the atoms under the operations, modulo:
- Rational equivalence (algebraic relations between cycles)
- The ring structure of CH^*(X) (intersection product)

The cycle class map: cl: C(X) ⊗ Q → H^*(X, Q)

**Hodge conjecture**: Im(cl) ⊇ Hdg^*(X)

## The Word Problem Formulation

**Definition**: A CYCLE WORD of complexity ≤ k is an expression in C(X)
built from atoms using ≤ k operations.

**The Hodge word problem**: For a given variety X and Hodge class α ∈ Hdg^p(X),
does there exist a cycle word w of finite complexity with cl(w) = α?

**If the word problem is SOLVABLE for X**: Hodge holds for X.

### Decidability

For FIXED X with known cohomology:
1. The target space Hdg^p(X) is finite-dimensional (say dim = ρ)
2. The cycle class map is a LINEAR MAP to a finite-dim Q-vector space
3. Each cycle word w evaluates to a specific vector cl(w) ∈ H^{2p}(X, Q)
4. We need ρ linearly independent vectors in {cl(w) : w a cycle word}

**Algorithm**:
```
rank = 0
for k = 1, 2, 3, ...:
    for each cycle word w of complexity k:
        v = cl(w)  # compute cycle class
        if v is linearly independent of previously found classes:
            rank += 1
            if rank == ρ:
                return "Hodge verified" ✓
```

This algorithm TERMINATES if Hodge is true (eventually finds ρ independent
classes). It RUNS FOREVER if Hodge is false (never reaches rank ρ).

**This is a SEMI-DECISION PROCEDURE**: it can verify Hodge but not refute it.

## Making It Fully Decidable: The Degree Bound Conjecture

**Conjecture (implicit in the literature)**: For a smooth projective
variety X of degree d in CP^N, every algebraic Hodge class can be
represented by a cycle of degree ≤ f(d, N) for some computable function f.

If this conjecture holds: the cycle search terminates in FINITE time
(only finitely many cycles of bounded degree). The algorithm becomes
a DECISION PROCEDURE.

For abelian varieties: the MT classification gives the degree bound
(the needed cycles come from endomorphisms, which have bounded degree
in terms of the MT group).

For cubic fourfolds: the Hassett classification gives the degree bound
(special surfaces in X have degree bounded by the discriminant).

## The Generator as a Formal System

### Level 0: Atomic Cycles
On X ⊂ CP^5 (cubic fourfold):
- h = hyperplane class (degree 1)
- h² = intersection of two hyperplanes (degree 1)
- Linear subspaces: P^2 ∩ X (if contained in X)

### Level 1: Simple Constructions
- Products: h · Z for any cycle Z
- Residual intersections: if S₁ ∩ S₂ = Z ∪ R, then R is a new cycle
- Projections: if X contains a line ℓ, project from ℓ to get a conic bundle

### Level 2: Correspondences
- If f: X → Y is a rational map, use the graph Γ_f ⊂ X × Y
- γ_* : CH(Y) → CH(X) produces new cycles from known cycles on Y
- Example: Fano variety of lines F(X) → X gives correspondences

### Level 3: Derived Constructions
- Chern classes of the universal bundle on F(X)
- Mukai vectors from derived categories
- K-theoretic constructions (Adams operations, γ-filtration)

### The Tree Search

```
                    HODGE CLASS α
                         │
          ┌──────────────┼──────────────┐
          │              │              │
     Level 0        Level 1        Level 2
    (atoms)       (products)   (correspondences)
     h, h², P²    h·Z, residual   γ_*(·), Fano
          │              │              │
     evaluate       evaluate       evaluate
     cl(·)          cl(·)          cl(·)
          │              │              │
          └──────────────┼──────────────┘
                         │
                  Span Hdg^p(X)?
                    /        \
                  YES          NO
                   │            │
              CERTIFICATE   INCREASE LEVEL
```

## Application: Cubic Fourfolds

For a cubic fourfold X ⊂ CP^5 with discriminant d:

**The Hodge lattice**: H^{2,2}(X) ∩ H^4(X, Z) ≅ ⟨h², S_d⟩ where S_d
is the "special" surface class (if it exists).

**Level 0**: h² is always algebraic (hyperplane section squared). ✓

**Level 1**: If X contains a surface S (e.g., a plane P² ⊂ X, a cubic
scroll, a Veronese surface), then [S] ∈ H^{2,2} is algebraic.
The class [S] mod Q·h² is the special class.

**The search**: enumerate surfaces in X by degree:
- degree 1: planes (if X contains a plane → d = 8)
- degree 2: quadric surfaces (if contained → d = 12)
- degree 3: cubic scrolls (→ d = 14)
- degree 4: quartic scrolls, del Pezzo (→ d = 20, 26, ...)
- ...

For each discriminant d: there's a SPECIFIC surface type that produces
the needed Hodge class. The Hassett classification predicts which d's
are "special" and what surface gives the extra class.

**The generator for cubics**: enumerate d, construct X_d, find the surface.
Certificate for each d. This covers a COUNTABLE set of varieties.

## The Meta-Theorem

**Theorem (structure of the generator)**:

For any ENUMERABLE family of varieties {X_t}:
1. The Hodge classes on X_t are COMPUTABLE (from periods)
2. The cycle algebra C(X_t) is ENUMERABLE (by complexity)
3. The cycle class map cl: C(X_t) → H*(X_t, Q) is COMPUTABLE
4. The spanning check rank(Im(cl)) = rank(Hdg) is DECIDABLE

Therefore: Hodge for each X_t is SEMI-DECIDABLE.

If a degree bound conjecture holds: Hodge for each X_t is DECIDABLE.

**The Hodge conjecture is a statement about ALL smooth projective varieties.
The generator can verify it for each one individually. The GENERAL statement
requires an argument that works for ALL varieties simultaneously — this is
what the MT classification provides for abelian varieties (it's uniform
in the variety, not case-by-case).**
