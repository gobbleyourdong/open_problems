## THEOREM

For N = 4 divergence-free Fourier modes on a SINGLE shell (all |k_j| = K)
on T³, at the argmax of |ω|²:

    Q/|ω|² ≥ min_{t>0} (3t² - 16K²t + 64K⁴)/(8K²t) > 0

where t = |ω|²/K². The discriminant 16²K⁴ - 4·3·64K⁴ = -512K⁴ < 0
means the polynomial has no real roots: Q > 0 for ALL |ω|².

The minimum Q/|ω|² on the shell: at t = 8K²/3:
    Q/|ω|² = (3·64/9 - 128/3 + 64)·K²/(8·8K²/3) = ... > 0 (positive constant)

## SIGNIFICANCE

The Key Lemma extremizers (Q/|ω|² → 0) require MIXED SHELLS:
modes at different wavenumbers. On any single shell, Q/|ω|² has a
POSITIVE lower bound depending only on N (not on K).

Near blowup: the energy cascade spreads energy across many shells.
If the DOMINANT modes are on a single shell (or few shells):
Q/|ω|² is bounded away from 0 → stronger bound → possible improvement.

If the modes are on MANY shells: N_eff is large → SOS data shows
c(N_eff) → 0 for bounded K → but unbounded K approaches Key Lemma.

The COMPETITION: near blowup, modes spread across shells (weakening Q)
but also increase N (strengthening Q via SOS floor growth).

## IMPLICATION FOR DYNAMICAL RIGIDITY

The Key Lemma extremizers (Q/|ω|² ≈ 0) require:
1. |ω|² near the discriminant root N²K²/3 (borderline)
2. Mixed shells (different |k| values)
3. Specific polarization alignment

The NS dynamics near blowup creates a specific spectral structure
(energy cascade, Kolmogorov-type spectrum). This structure may be
INCOMPATIBLE with the extremizer requirements.
