## THEOREM

For any N ≤ 4 divergence-free modes on T³ with unit amplitudes,
at x* = argmax|ω|²:

    S²ê(x*) < (3/4)|ω(x*)|²

## PROOF

### Step 1-3 (same as file 726)

Argmax: |cosγⱼ| ≥ 1/|ω| → sinγⱼ ≤ √(1-1/|ω|²).
Self-vanishing + triangle + CS: S²ê ≤ N²(|ω|²-1)/(4|ω|²).
Need: N²(|ω|²-1) < 3|ω|⁴, i.e., **3t² - N²t + N² > 0** where t = |ω|².

### Step 4a (N ≤ 3): Discriminant < 0.
N²(N²-12) < 0 for N ≤ 3. No real roots. Polynomial always positive. ∎

### Step 4b (N = 4): The R³ dimension argument.

The polynomial 3t² - 16t + 16 has roots t = 4 and t = 4/3.
Positive for t > 4 (and t < 4/3). Need: **|ω|² > 4 at the argmax.**

**Lemma**: For N ≥ 4 unit vectors in R³: max_s |ω_s|² > N strictly.

*Proof of Lemma*: max_s |ω_s|² = N requires all Dᵢⱼ = vᵢ·vⱼ = 0
(isolate each D by choosing sign patterns). This means N pairwise
orthogonal unit vectors in R³. But dim(R³) = 3 < 4 ≤ N. Impossible. □

So |ω|² > 4 = N at the argmax. The polynomial 3t²-16t+16 is positive
for t > 4. Therefore S²ê < (3/4)|ω|². ∎

### Why it fails for N ≥ 5

For N=5: roots of 3t²-25t+25 are t ≈ 7.17 and t ≈ 1.16.
Need |ω|² > 7.17. But |ω|² could be as low as ~5 (just above N).
5 < 7.17: the polynomial is negative. Proof fails.

The R³ argument gives |ω|² > N but not |ω|² > N²/3 (the larger root).
