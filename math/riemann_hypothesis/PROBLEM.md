# Riemann Hypothesis

## Statement
Every non-trivial zero of the Riemann zeta function ζ(s) has real part 1/2.

## Status: OPEN (Clay Millennium Prize, $1M)

## Key Objects
- ζ(s) = Σ n^{-s} for Re(s) > 1, analytically continued to C \ {1}
- Functional equation: ζ(s) = 2^s π^{s-1} sin(πs/2) Γ(1-s) ζ(1-s)
- Critical strip: 0 < Re(s) < 1
- Critical line: Re(s) = 1/2
- Trivial zeros: s = -2, -4, -6, ...
- Non-trivial zeros: all known ones lie on Re(s) = 1/2

## Known Results
- ~10^13 non-trivial zeros verified on the critical line (numerical)
- Zero-free region: Re(s) > 1 - c/log(|Im(s)|) (de la Vallée-Poussin)
- 40%+ of zeros on critical line (Conrey, 1989)
- Equivalent to: Li(x) - π(x) = O(x^{1/2} log x)
- GRH (generalized) implies many results in number theory

## Approaches
- Analytic number theory (zero-free regions, density estimates)
- Random matrix theory (GUE statistics of zero spacings)
- Spectral theory (Hilbert-Pólya: self-adjoint operator whose eigenvalues are zeros)
- Arithmetic geometry (Weil conjectures analog for number fields)
