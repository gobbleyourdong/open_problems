# Attempt 001: T1DM Protocol Adapted for ME/CFS

## Approach
Adapt the converged T1DM protocol (THEWALL.md) for ME/CFS, accounting for the unique constraint of post-exertional malaise (PEM). The core is identical — clear CVB, suppress inflammation, restore immune regulation — but the order and intensity differ.

## Critical Adaptation: PEM-Safe Sequencing

In T1DM, the protocol can be started simultaneously (everything is oral/dietary).
In ME/CFS, **metabolic stressors must be introduced gradually** because the crash threshold is unknown.

### Phase 0: Baseline + Stabilize (Weeks 1-2)
- Establish baseline: SF-36, 2-day CPET, NK cell assay, cytokine panel
- Start ONLY non-stressor interventions:
  - Vitamin D 5000 IU daily
  - Omega-3 (EPA/DHA) 3g daily
  - Butyrate 300mg TID (gut barrier + Tregs)
  - GABA 750mg daily (anti-inflammatory + neurotransmitter support)
  - Magnesium glycinate 400mg daily
- **No fasting, no cold exposure, no exercise yet**

### Phase 1: Antiviral (Weeks 3-6)
- Add fluoxetine 20mg daily (start at 10mg, titrate)
  - Bonus: SSRI effect may improve ME/CFS depression/anxiety component
  - Fluoxetine has 5-7 day half-life — steady state at ~4 weeks
- Continue all Phase 0 supplements
- Monitor: energy levels, crash frequency, sleep quality

### Phase 2: Autophagy Activation (Weeks 7-10)
- Introduce time-restricted eating (16:8) first — less intense than FMD
- If tolerated for 2 weeks: first FMD cycle (5-day fast-mimicking diet)
  - **Critical**: during FMD, absolute rest. No activity. The metabolic stress of fasting + autophagy activation is the therapeutic load.
- WHM breathing exercises (respiratory only — NOT cold exposure yet)
  - Intermittent hypoxia → HIF-1α → autophagy pathway
  - No muscular exertion involved

### Phase 3: Cold Exposure (Weeks 11-14)
- Start cold exposure ONLY after first FMD cycle shows no crash
- Begin with cold face immersion (30 seconds)
- Progress to cold shower (30 seconds → 1 minute → 2 minutes)
- Target: norepinephrine spike → IL-10 production → anti-inflammatory
- **If any cold exposure triggers PEM: stop, reassess, wait 2 weeks**

### Phase 4: Reintroduce Activity (Weeks 15+)
- **ONLY after documented improvement in NK cell function or symptom scores**
- Start with isometric exercises (no eccentric muscle loading)
- Progress to walking (5 minutes → 10 minutes → 15 minutes)
- 2-day CPET retest at week 16 to document change in anaerobic threshold

## What This Solves
- Viral persistence (fluoxetine + autophagy/FMD)
- Neuroinflammation (BHB from fasting, GABA, omega-3)
- Gut dysbiosis (butyrate)
- Immune dysregulation (vitamin D → Tregs, cold exposure → IL-10)
- NK cell dysfunction (autophagy may restore function — needs testing)

## What This Doesn't Solve
1. **Non-CVB ME/CFS** — if the patient's ME/CFS isn't CVB-driven, antiviral components won't help (but anti-inflammatory stack still benefits)
2. **Autonomic dysfunction** — POTS/orthostatic issues may need separate management (compression, fludrocortisone, midodrine)
3. **Sleep architecture** — unrefreshing sleep may need targeted intervention (low-dose naltrexone? melatonin?)
