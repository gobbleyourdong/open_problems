## Import 1: M1↔M8 HPA Exhaustion → Explains ME/CFS Hypocortisolism

### Background — the ME/CFS cortisol paradox:

ME/CFS is characterized by HYPOCORTISOLISM (low cortisol), not hypercortisolism.
This is the opposite of acute stress response. Standard explanation: "adrenal insufficiency"
from unclear cause. Many theories; no mechanistic consensus.

### The dysbiosis framework explanation:

The M1↔M8 bidirectional loop (dysbiosis/attempts/attempt_016 + attempt_015) runs as follows:

**Phase 1 — Ascending (hypercortisolism phase):**
```
CVB infection (acute ME/CFS trigger) → gut dysbiosis (M1 active)
    ↓ M1→M8 pathway (bottom-up):
Gut dysbiosis → reduced SCFA (F. prausnitzii depleted) → reduced GPR41/43 on gut EC cells →
reduced PYY/GLP-1 → HPA hypersensitivity (Sudo 2004 GF mouse model)
    ↓
SIMULTANEOUSLY: Gut LPS → enteroendocrine CRF → primes HPA directly from gut
    ↓
HPA → elevated cortisol CHRONICALLY → elevated for months-years in early ME/CFS
    ↓
During this phase: cortisol is an IMMUNE SUPPRESSANT → TH1 impaired → viral persistence facilitated
→ this actually makes ME/CFS worse (more CVB persistence as cortisol suppresses NK function)
```

**Phase 2 — Plateau and Descent (transition to hypocortisolism):**
```
Chronic HPA overstimulation → glucocorticoid receptor (GR) downregulation in HPA feedback nodes:
    Hippocampus GR downregulation → impaired negative feedback → HPA runs higher for longer
    Hypothalamic CRH neurons: prolonged activation → neuronal fatigue
    Adrenal cortex: prolonged ACTH stimulation → eventual adrenal fatigue (controversial but documented)
    ↓
After months-years of M1-driven chronic HPA activation → GR-mediated negative feedback fails →
Paradoxically, HPA output FALLS (blunted ACTH response to CRH; blunted cortisol to ACTH)
    ↓
END STATE: HYPOCORTISOLISM — not the start state but the EXHAUSTION STATE of the M1↔M8 loop
```

**This mechanistically explains the ME/CFS cortisol profile:**
- Early ME/CFS (onset to ~2-3 years): sometimes normal or slightly elevated cortisol
- Established ME/CFS (>3-5 years): consistently low cortisol (documented in Cleare 2004 review)
- Severe ME/CFS: profound hypocortisolism, orthostatic intolerance (cortisol needed for vasomotor tone)

**The M6 × M8 interaction (relevant for pediatric-onset ME/CFS):**
If early-life dysbiosis (low M6 floor → reduced HPA dampening capacity from early-life SCFA deficit)
precedes CVB infection → HPA is ALREADY hypersensitive at baseline → ME/CFS trigger more easily
causes HPA exhaustion → faster progression to hypocortisolism in pediatric/adolescent ME/CFS.

### Clinical implication:

If M1-driven HPA exhaustion is the mechanism, the intervention is:
1. Fix gut dysbiosis (M1) first → reduces the chronic HPA stimulation from the gut CRF axis
2. SCFA restoration (butyrate + fiber) → restores GPR41/43 → HPA dampening
3. Avoid pharmacological cortisol replacement as primary treatment (does not address root cause;
   may suppress remaining adrenal function)
4. Monitor: as M1 is treated, cortisol may transiently RISE before normalizing (HPA recovering
   from suppression) — this can be alarming if not expected

## Import 2: Three Non-Responder Loops Applied to ME/CFS Chronification

The dysbiosis campaign identified three independent sustaining loops that create a non-responder
state in rosacea. The same three loops operate in ME/CFS:

### Loop 1 equivalent: KLK5/IFN/IL-23 → ME/CFS version: IFN/STAT/NK exhaustion

In ME/CFS, the IFN signature (STAT1/STAT2 upregulation from THEWALL.md) is a self-sustaining loop:
```
CVB (persistent) → IFN-α → STAT1 ↑ → ISG expression ↑ → T cell exhaustion (PD-1, Tim-3 ↑)
    ↓
IFN-α sustained → NK activation → PRF1 ↑, granzymes ↑ (NK armed but cannot kill TD mutants)
    ↓
Futile NK response → NK degranulate without targets → NK exhaustion → NK dysfunction
    ↓
Less NK-mediated viral pressure → CVB persists more easily → more IFN-α
```

**Loop 1 ME/CFS non-responder break:** Reduce CVB persistence (FMD + fluoxetine for CVB-positive)
+ reduce futile NK activation (treatment unclear; possibly reducing IFN-α will reduce NK signaling).

### Loop 2 equivalent: NLRP3/pyroptosis → documented in ME/CFS (NLRP3 +37% in THEWALL.md)

```
CVB 2B viroporin → K+ efflux → NLRP3 activation (exactly as in rosacea Loop 2)
    ↓
IL-1β → neuroinflammation → microglial NLRP3 activation → brain fog/fatigue
    ↓
Pyroptosis → DAMPs → re-prime NLRP3 (same DAMP cycle as rosacea)
    ↓
BHB + colchicine + IF = SAME treatment as rosacea Loop 2 (mechanisms are identical)
T1DM hyperglycemia primes NLRP3 — equally relevant to ME/CFS in T1DM patients
```

**Loop 2 ME/CFS non-responder break:** BHB + colchicine + IF. Already in ME/CFS THEWALL.md
(BHB/CoQ10 are there; colchicine is NOT — this is an import). Add colchicine 0.5mg BID
for ME/CFS patients with elevated IL-18 or NLRP3 markers.

### Loop 3 equivalent: HERV-W self-sustaining → particularly relevant in EBV-triggered ME/CFS

```
EBV reactivation (common ME/CFS trigger) → HERV-W activation (EBV → HERV-W epigenetic unsilencing)
    ↓
MSRV-Env → TLR4 → NF-κB → IL-6/TNF-α
    ↓
IL-6/TNF-α → NF-κB in HERV-W promoter → HERV-W sustained
    ↓
HERV-W → additional IFN-β → IFN-α cascade → amplifies the STAT/IFN signature of ME/CFS
    ↓
EBV is cleared but HERV-W continues → EBV-triggered ME/CFS "persists" after EBV resolution
```

**Loop 3 ME/CFS non-responder break:** same as rosacea — gut/sleep protocol + colchicine
(NF-κB suppression). Anti-MSRV-Env (temelimab) is not available but would be the specific intervention.

**Critical clinical implication:** EBV-triggered ME/CFS patients (positive EBV serology)
who do NOT respond to antivirals (valacyclovir) may have HERV-W as the sustaining mechanism —
EBV is gone but HERV-W loop runs. Valacyclovir targets EBV but not HERV-W.

## Import 3: TRPV1 Neurogenic Sensitization → ME/CFS Central Sensitization

The TRPV1 axis (dysbiosis/numerics/run_015) is directly relevant to ME/CFS post-exertional malaise (PEM):

```
ME/CFS: IFN-α elevated chronically → TRPV1 upregulation in sensory neurons (CNS and peripheral)
    ↓
SP (Substance P) from neurogenic inflammation (M8 equivalent: CRH/cortisol/HPA dysregulation)
    → TRPV1 sensitization: threshold lowered to body temperature
    ↓
ANY EXERTION → heat generation → TRPV1 activation → widespread neurogenic pain/fatigue signal
    → Post-exertional malaise mechanism:
       Physical exertion → body temperature ↑ → sensitized TRPV1 fires throughout CNS/muscle →
       widespread pain, fatigue, cognitive dysfunction
```

**This is a new mechanistic model for PEM** — not "mitochondrial ATP deficit" alone (which THEWALL.md
already has) but ALSO TRPV1-mediated central sensitization where any heat-producing activity
triggers disproportionate TRPV1-driven pain/fatigue signaling.

**Intervention implication:** Low-dose naltrexone (LDN) — already used off-label in ME/CFS —
partially reduces microglial TRPV1 overexpression via opioid receptor normalization. This
provides a mechanistic explanation for why LDN helps some ME/CFS patients beyond "microglial
activation."

## Import 4: SPM/Resolvin DAMP Clearance → ME/CFS Tissue Inflammation Resolution

The SPM analysis (dysbiosis/numerics/run_020) applies directly to ME/CFS post-viral tissue damage:

CVB → pyroptosis in muscle/gut/cardiac tissue → DAMPs (HMGB1, ATP, uric acid) accumulate →
NLRP3 re-primed → Loop 2 sustains → tissue never fully resolves.

RvD1 (DHA-derived) + RvE1 (EPA-derived) at 4g/day omega-3:
- Clears HMGB1 from post-pyroptotic tissue via macrophage efferocytosis
- Reduces NLRP3 re-priming substrate
- Promotes macrophage M2 reprogramming → anti-inflammatory mediators → tissue healing

**For ME/CFS muscle tissue damage (MT-ND3 damage, mitochondrial dysfunction):**
Omega-3 at 4g/day → PD1 (neuroprotectin D1) → protects mitochondrial membranes + reduces
neurogenic inflammation in dorsal root ganglia → supports mitochondrial recovery alongside
CoQ10 + NAD+ precursors.
