# Pericarditis — Phase 2 Summary (After 2 Attempts)

| 001 | Colchicine + fluoxetine RCT | Add antiviral to standard of care | Trial designed: 3-arm, n=195, 18 months, binary endpoint |
| 002 | Recurrence mechanism model | Why colchicine alone fails 30% | NLRP3 cycle formalized; colchicine masks, doesn't cure |

## What We Know Now

1. **Pericarditis is the BEST proof-of-concept disease for the entire campaign.** Binary endpoint (recurrence y/n), cheap generic drugs, existing standard of care to add to, feasible sample size. If we can only run ONE trial, this should be it.

2. **The NLRP3 model explains the 30% recurrence rate perfectly.** Colchicine blocks NLRP3 activation (Signal 2) but doesn't clear the virus. When colchicine is stopped, persistent CVB reactivates NLRP3 via dsRNA (Signal 1 was always primed). The 70% who don't recur happened to clear CVB on their own during the colchicine window.

3. **The key mechanistic prediction is DIFFERENTIAL recurrence after drug cessation.** If fluoxetine clears CVB, the intervention arm should show fewer recurrences AFTER stopping treatment (months 6-18). The control arm recurs when colchicine stops. This temporal pattern proves viral clearance, not just anti-inflammatory augmentation.

4. **The SSRI retrospective database query could provide signal TOMORROW.** Pericarditis patients already on fluoxetine for depression → do they recur less? Testable with existing insurance claims data (Optum, Truven). No new study needed.

## The Gap (Updated)
