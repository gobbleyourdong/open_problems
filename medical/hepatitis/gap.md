# CVB Hepatitis — Gap Analysis (Initial)

## The Gap

### Adult form: minimal gap
CVB hepatitis in adults is usually mild and self-limiting. Elevated ALT/AST, resolves in weeks. The gap is diagnostic — CVB is almost never tested in hepatitis workups (HBV/HCV/HAV/HEV are checked first, then autoimmune, then "cryptogenic").

### Neonatal form: overlaps with neonatal sepsis
Fulminant hepatic necrosis in neonates is part of the neonatal sepsis syndrome (see neonatal_sepsis/gap.md). The hepatic component is often the cause of death via coagulopathy.

## Practical Relevance to T1DM Protocol

**Drug metabolism concern.** If the patient (or any CVB-protocol patient) has subclinical CVB hepatitis:
- Itraconazole is hepatotoxic and hepatically metabolized
- Fluoxetine is hepatically metabolized (CYP2D6)
- Liver function must be checked BEFORE starting the protocol

**Baseline labs for any CVB protocol patient should include:**
- ALT, AST, GGT, ALP, bilirubin (total + direct)
- If elevated: investigate before starting hepatotoxic drugs

## Anti-Problem
"Is subclinical CVB hepatitis present in T1DM patients?"

If persistent CVB infects the pancreas, does it also infect the liver? The liver receives portal venous blood directly from the pancreas. CVB draining from infected islets flows directly to the liver.

**Prediction**: T1DM patients with persistent CVB have mildly elevated ALT/AST that has been attributed to fatty liver, metabolic syndrome, or "normal variation" — but is actually subclinical CVB hepatitis.

## Protocol
No disease-specific protocol needed beyond the T1DM protocol + liver function monitoring. The liver regenerates well — clearing CVB is sufficient for recovery.
