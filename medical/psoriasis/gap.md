# Psoriasis — Gap Analysis (Initial)

## How Psoriasis Fits the Campaign

Like eczema, psoriasis is NOT a CVB disease. But it overlaps with the campaign's targets even MORE than eczema does:

```
CVB diseases:  Virus → NF-κB → TNF-α → organ damage
Psoriasis:     Autoantigen (LL-37) → NF-κB → TNF-α → IL-23 → Th17 → skin damage

SHARED TARGET: NF-κB / TNF-α axis
The T1DM arsenal (attempt 062) attacks this axis at 7 points.
All 7 apply to psoriasis.
```

## The Gap

### For psoriasis specifically
The gap is the **IL-23/Th17 axis**. NF-κB and NLRP3 are amplifiers, but the core driver is IL-23 maintaining Th17 cells. The protocol's anti-inflammatory stack dampens the amplification but may not hit the IL-23 driver directly.

- BHB, colchicine, WHM → suppress NLRP3, NF-κB (the amplifiers) ✅
- Butyrate, vitamin D → induce Tregs that suppress Th17 (indirect Th17 suppression) ✅
- Omega-3 → resolvins (anti-inflammatory) ✅
- **Nothing in the protocol directly blocks IL-23 or IL-17** ❌
- Biologics (guselkumab, secukinumab) block these directly but cost $30K-60K/year

### The question
Can suppressing the AMPLIFIERS (NF-κB, NLRP3) + restoring BRAKES (Tregs) reduce psoriasis enough without directly blocking the DRIVER (IL-23/Th17)?

- If yes: the protocol provides 50-70% improvement in psoriasis as a side effect
- If no: psoriasis needs IL-23 or IL-17 blockade (biologics) on top of the protocol

## The Apremilast Connection

Apremilast (Otezla) is the bridge drug between T1DM and psoriasis:
- FDA-approved for psoriasis and psoriatic arthritis
- PDE4 inhibitor → cAMP → PKA → CREB → IL-10 ↑, TNF-α ↓, IL-17 ↓, IL-23 ↓
- $30-60/month generic (off-patent soon)
- T1DM attempt 062 identified it as the "real" PDE4 inhibitor (vs caffeine)
- **Adding apremilast to the protocol would address psoriasis AND enhance T1DM treatment**

```
Protocol + apremilast:
  T1DM benefit: PDE4 → cAMP → IL-10 ↑, TNF-α ↓ (additive to WHM/BHB/butyrate)
  Psoriasis benefit: FDA-approved indication, directly suppresses IL-17/IL-23/TNF-α
  Cost: $30-60/month (approaching generic)
  Side effects: GI (nausea, diarrhea — usually transient), weight loss
```

## The Treg/Th17 Balance

Tregs and Th17 cells differentiate from the same naïve CD4+ precursor. The balance is determined by the cytokine environment:

```
TGF-β alone → Treg (FOXP3+) → immune suppression
TGF-β + IL-6 → Th17 (RORγt+) → immune activation (psoriasis)
TGF-β + IL-1β → Th17 → immune activation

NLRP3 → IL-1β → Th17 differentiation
BHB suppresses NLRP3 → less IL-1β → less Th17 → MORE Treg
Butyrate → FOXP3 → more Treg → less Th17

THE PROTOCOL SHIFTS THE TREG/TH17 BALANCE TOWARD TREG.
This is the mechanism by which it should improve psoriasis without directly blocking IL-23/IL-17.
```

## Priority

Like eczema: quality-of-life disease, not life-threatening. If the protocol improves it as a side effect, that's valuable evidence for the shared NF-κB/NLRP3/Treg mechanism. Track PASI score alongside other metrics — it's free data.

## Convergence with Eczema

| Feature | Eczema | Psoriasis |
|---------|--------|-----------|
| Immune driver | Th2 (IL-4/IL-13) | Th17 (IL-17/IL-23) |
| NLRP3 involvement | Yes | Yes |
| NF-κB involvement | Moderate | Strong |
| Gut-skin axis | Yes | Yes |
| Vitamin D responsive | Yes | Yes (topical is standard Rx) |
| Omega-3 responsive | Modest | Modest |
| Butyrate/Treg relevant | Yes (Treg suppresses Th2) | Yes (Treg suppresses Th17) |
| Protocol predicted benefit | 30-50% improvement | 30-50% improvement |

**Both diseases improve via the same mechanism: Treg restoration suppresses the pathogenic T helper subset (Th2 or Th17).** Butyrate + vitamin D + BHB → Tregs → suppress both.
