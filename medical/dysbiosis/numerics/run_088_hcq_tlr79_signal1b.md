## Part A: Hydroxychloroquine → TLR7/TLR9 → Signal 1B ↓

### HCQ Mechanism: Endosomal pH Elevation

**TLR7/TLR9 require acidic endosomes:**
```
TLR7: recognizes single-stranded RNA (ssRNA) in endosomes
    → Activated by: viral ssRNA (CVB, SARS-CoV-2, HERV-W RNA fragments)
    → pDC endosomal TLR7 → MyD88 → IRF7 → IFN-α
TLR9: recognizes unmethylated CpG DNA in endosomes
    → Activated by: mitochondrial DNA (mtDNA), NET-DNA/LL-37 complex (run_081)
    → pDC endosomal TLR9 → MyD88 → IRF7 → IFN-α (Lande 2007 lupus mechanism)
    
Both TLR7/9 require endosomal acidification (pH 4.5-6.0) for:
    → Nucleic acid ligand processing (RNA/DNA degradation to active fragments)
    → Conformational change of TLR extracellular domain (pH-sensitive)
    → Recruitment of signaling adaptors in endosomal membrane
```

**HCQ mechanism (lysosomal accumulation → pH ↑):**
```
Hydroxychloroquine → weakly basic (pKa 8.3 and 9.7)
    → Protonated at physiological pH → traps in acidic compartments (lysosomes, endosomes)
    → Accumulates to high concentrations (>100× plasma concentration in lysosomes)
    → Lysosomal/endosomal pH ↑ from 4.5-5.0 → 5.5-6.5 (partial alkalinization)
    ↓
TLR7/9 cannot activate:
    → Nucleic acid ligands not properly processed (pH-dependent cleavage requires acid)
    → TLR conformational change does not occur
    → MyD88 recruitment ↓ → IRF7 → IFN-α NOT produced
    ↓
Signal 1B (ISGF3/IFN-α priming of NLRP3): specifically suppressed at SOURCE
    (HCQ acts on TLR7/9 in pDC endosomes → IFN-α production ↓, not receptor)
```

**Evidence in IFN-α reduction:**
```
Lupus (SLE): TLR7/TLR9 → IFN-α → SLE pathology. HCQ → IFN-α ↓ (Visvanathan 2013 
    Arthritis Rheum: HCQ treatment → IFN-α ↓ in SLE patients)
T1DM observational:
    Wasko 2012 JAMA 307:1030: RA patients on HCQ → T2DM incidence ↓ 50-77%
    Yao 2016 Diabetes Care: T2DM/T1DM on HCQ → HbA1c ↓ 0.5-0.7% (unexpected glucose-lowering)
    Potential mechanism: TLR9 → IFN-α → β cell IFNAR → MHC-I ↑ → CTL recognition ↑ → β cell
    destruction. HCQ → TLR9/IFN-α ↓ → β cell stress ↓ → better C-peptide preservation.
```

### HCQ in the Framework: Signal 1B Direct Suppressor

**Node D high patients (IFN-α2 Simoa >0.05 fg/mL) — Signal 1B-dominant:**
```
Standard NF-κB protocol (ten suppressors) targets Signal 1A.
Signal 1B (ISGF3/IFN-α): no direct target in standard protocol
    → M3 management (HERV-W, CVB): indirect; viral/retroviral sources suppressed over months
    → Run_063 (niacinamide/PARP-1/CPD): UV source of Signal 1B addressed
    → Run_081 (NETs): TLR9/pDC/IFN-α source; colchicine addresses NETosis upstream
    → HCQ: DIRECT endosomal TLR7/9 block → IFN-α ↓ regardless of source

HCQ is the most direct Signal 1B suppressor available:
    Blocks TLR7 (HERV-W RNA, CVB ssRNA) + TLR9 (NET-DNA, mtDNA, viral DNA)
    → All nucleic acid-based Signal 1B inducers addressed simultaneously by one agent
    → Position: specialist-adjunct for persistently elevated Node D (IFN-α) despite full protocol
```

**Safety and T1DM context:**
```
HCQ safety:
    Retinal toxicity: cumulative dose >5 mg/kg/day × >5 years → maculopathy risk
    Recommended screening: annual ophthalmology after 5 years of use
    Cardiac: QTc prolongation (baseline ECG; avoid with other QTc-prolonging agents)
    
T1DM compatibility:
    Glucose-lowering effect: may require insulin dose reduction (hypoglycemia risk)
    → Close glucose monitoring in first 4-8 weeks after initiation
    Autoimmune overlap: T1DM patients with concurrent SLE/lupus traits → HCQ already used
    
Protocol position:
    Specialist-adjunct (rheumatology/immunology co-management)
    Node D persistently >0.05 fg/mL at two consecutive 6-month checks despite full protocol
    Dose: 200-400mg/day HCQ (standard lupus dose); adjusted by body weight (≤5 mg/kg/day)
    Duration: reassess Node D at 6 and 12 months; continue if Node D responding
```

## Part B: Hydroxytyrosol (EVOO) → SIRT1 + Nrf2

**Hydroxytyrosol: the most bioactive EVOO polyphenol:**
```
Extra virgin olive oil (EVOO) contains multiple phenols:
    Oleocanthal: COX-2 inhibitor (already in framework: omega-3/COX-2 context)
    Oleuropein: SIRT1 activator (weak)
    Hydroxytyrosol (HT; 3,4-dihydroxyphenylethanol): most potent antioxidant/SIRT1 activator
    → HT is the primary reason EVOO is more anti-inflammatory than refined olive oil
    → EVOO with >200 mg/kg HT content: high-phenol EVOO (Greek/Southern Italian varieties)
```

**HT → SIRT1 → NLRP3 K496 deacetylation:**
```
Hydroxytyrosol → SIRT1 activation (Oliva 2019 J Agric Food Chem; Parkinson 2014 JACS):
    → SIRT1 → deacetylates NLRP3 K496 → NLRP3 cannot adopt active conformation
    → Same mechanism as niacinamide mechanism 1 (run_031 SIRT1/NLRP3 context) and melatonin
    → Third dietary SIRT1 activator in framework (alongside quercetin/niacinamide/melatonin)
    
HT plasma concentrations from EVOO (2 tbsp/day = 25-30 mL EVOO with ≥200 mg/kg HT):
    → ~10-15 µg HT absorbed → plasma ~0.05-0.5 µM after digestion
    → Bioavailability: ~30-40% from EVOO matrix (olive oil matrix enhances absorption)
    → At µM range: SIRT1 activation confirmed (Parkinson 2014 JACS)
```

**HT → Nrf2 (KEAP1 alkylation):**
```
Hydroxytyrosol catechol → o-quinone oxidation product → electrophilic
    → KEAP1 Cys151/273 alkylation → Nrf2 released → anti-oxidant gene expression ↑
    → Same KEAP1/Nrf2 mechanism as sulforaphane (run_014 second NF-κB suppressor) + itaconate (run_084)
    → HT provides dietary Nrf2 activation from EVOO alongside sulforaphane from broccoli sprouts
    → Combined KEAP1 alkylators in protocol: sulforaphane + HT (diet) + CAPE propolis (p65 Cys38)
```

**Protocol note:**
```
Mediterranean diet EVOO recommendation: 2 tablespoons/day (25-30 mL) high-phenol EVOO
    → High-phenol EVOO: California Olive Ranch "Everyday" or Greek/Cretan EVOO (PDO varieties)
    → Check label: "polyphenol content" or "oleocanthal/hydroxytyrosol" if listed
    → Cooking at low-medium heat only (do not fry; polyphenols degrade above 180°C)
    → Raw use preferred (dressings, dipping): maximum HT preservation
    
Separate HT supplement: available (usually from olive leaf extract or olive fruit extract)
    → OTC; not widely studied for rosacea; EVOO food source preferred
```

## Kill Criteria

**Kill A: HCQ Glucose-Lowering Causes Hypoglycemia Risk in T1DM**
T1DM already on insulin + HCQ → unexpectedly lowered glucose → hypoglycemia.
**Status:** Real concern. Requires insulin dose reduction guidance on initiation. Frequent glucose monitoring in weeks 1-8. HCQ glucose-lowering magnitude: 0.5-0.7% HbA1c (mild) but clinically significant in tightly controlled T1DM. Not a kill; addressed by standard clinical caution. Endocrinology co-management required for HCQ in T1DM.

**Kill B: HCQ Maculopathy Risk at Rosacea-Duration Use**
HCQ for rosacea/T1DM (possibly years) → cumulative retinal dose risk. At ≤5 mg/kg/day × >5 years: ophthalmology monitoring required.
**Status:** Not a kill for short-term (≤5 years) use with appropriate screening. Long-term use: annual ophthalmology baseline + fundus examination. Risk is manageable with monitoring; does not prohibit use.

**Kill C: Hydroxytyrosol SIRT1 Activation Not Directly Confirmed in Dermal Macrophages**
Parkinson 2014/Oliva 2019 are in diverse cell types; dermal macrophage-specific HT-SIRT1 data lacking.
**Status:** Not killed. SIRT1 is expressed in all nucleated cells; HT-SIRT1 mechanism is not cell-type specific (SIRT1 protein structure is conserved). EVOO dietary intake is safe and recommended regardless; this is an additional mechanistic rationale for an already-recommended dietary pattern.
