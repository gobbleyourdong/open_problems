## TRPA1 Biology and Skin/Sensory Neuron Expression

**TRPA1 structure and activation:**
```
TRPA1 (TRPA1 gene): 1119 amino acid, 6-transmembrane TRP channel
    → Expressed in: DRG neurons (sensory), trigeminal ganglion neurons, keratinocytes, mast cells
    → Co-expressed with TRPV1 on the same C-fiber and Aδ-fiber nociceptors
    → Activation → Ca²⁺ influx + Na⁺ influx → depolarization → action potential
    → At nerve terminals: Ca²⁺ → exocytosis of SP (substance P) + CGRP (calcitonin gene-related peptide)
    
Activation mechanism for electrophiles (4-HNE, cinnamaldehyde):
    Michael acceptors: 4-HNE, cinnamaldehyde, allicin → COVALENTLY modify TRPA1 Cys621, Cys641, Cys665
    → Cys modification → conformational change → channel open
    → This is IRREVERSIBLE activation until receptor internalized or Cys reduced
    → Unlike TRPV1 (reversible thermal gating), electrophile-activated TRPA1 is persistent
    
H₂O₂ activation: oxidizes Met and Cys residues → TRPA1 activation (redox-sensitive)
Methylglyoxal: dicarbonyl → Arg/Lys modification near Cys sites → TRPA1 gating
```

**TRPA1 in rosacea-specific tissue:**
```
Buhl 2017 J Invest Dermatol: TRPA1 immunoreactivity in facial skin nerve fibers of rosacea patients
    → TRPA1+ nerve fiber density ↑ in rosacea lesional skin vs. controls
    → Co-localization with SP and CGRP on same fibers (confirm sensory neuropeptide source)
    
Mascarenhas 2019 J Dermatol: TRPA1 mRNA ↑ in rosacea biopsies (erythematotelangiectatic type)
    → Elevated TRPA1 expression relative to non-rosacea controls
    → Consistent with TRPA1 as an active driver, not merely a passive sensor
    
TRPV1 (run_015) and TRPA1 relationship:
    Both expressed on same sensory neurons → SYNERGISTIC activation
    TRPV1 activation (heat, capsaicin) → sensitizes TRPA1 to sub-threshold electrophile concentrations
    TRPA1 activation → sensitizes TRPV1 via shared intracellular Ca²⁺ and PKA pathways
    → Called "sensitization cross-talk": one channel activation lowers threshold of the other
    → In rosacea dermis: chronic inflammation → TRPV1 sensitized → now TRPA1 triggers at lower 4-HNE/MG concentrations
```

## TRPA1: The Loop 4 → M8 Bridge

**4-HNE (4-hydroxynonenal): the primary endogenous TRPA1 agonist in rosacea:**
```
Loop 4 mechanism (established; runs 025, 082):
    UV + Malassezia lipases → squalene oxidation → squalene-OOH (squalene peroxide)
    Squalene-OOH → secondary peroxidation products:
        4-HNE (4-hydroxynonenal): α,β-unsaturated aldehyde, Michael acceptor
        MDA (malondialdehyde): similar electrophile
    4-HNE → NLRP3 Signal 2 (lysosomal disruption; established in Loop 4 context)
    
NEW TRPA1 connection (run_093):
    4-HNE → TRPA1 Cys621/641/665 covalent modification → TRPA1 gating → Ca²⁺ influx →
    → Sensory neuron depolarization → SP/CGRP release → dermal vasodilation → rosacea flush
    
Evidence: Trevisani 2007 PNAS 104(33):13519-13524: 4-HNE → TRPA1 activation confirmed
    → IC50 ~10-20 µM in sensory neurons; rosacea skin 4-HNE concentrations estimated 5-50 µM
    → 4-HNE generated from Loop 4 is in TRPA1-activating concentration range
    
COMPLETE LOOP 4 → M8 PATHWAY:
    UV → squalene peroxide → 4-HNE → [A] NLRP3 Signal 2 (established) + [B] TRPA1 → SP/CGRP → M8
    → Loop 4 is BIFUNCTIONAL: drives both NLRP3 activation (Loop 2 amplification) AND neurogenic flushing (M8)
    → Previously these were disconnected; TRPA1 is the bridge
```

**H₂O₂ and TRPA1 in rosacea:**
```
Multiple H₂O₂ sources in rosacea dermis:
    Demodex/B. oleronius oxidative burst → H₂O₂
    NADPH oxidase (Ang II/RAAS; run_092) → O₂•⁻ → H₂O₂ via SOD1/2
    UV → keratinocyte ROS → H₂O₂
    
H₂O₂ → TRPA1 activation (Andersson 2011 PNAS; Takahashi 2011):
    H₂O₂ → Met412 sulfoxidation + Cys modification → TRPA1 open
    → Ca²⁺ → SP/CGRP → M8 activation
    
TRPA1 acts as a REDOX SENSOR:
    Any source of reactive oxygen → TRPA1 → neurogenic inflammation
    H₂O₂ from RAAS/Nox2 → TRPA1 = RAAS → M8 connection (via run_092's Nox2 products)
```

## T1DM Methylglyoxal → TRPA1: Diabetic Neurogenic Hyperactivation

**Methylglyoxal (MG) — T1DM-specific TRPA1 agonist:**
```
Methylglyoxal origin in T1DM:
    Hyperglycemia → ↑ triose phosphate (glycolysis intermediate) → methylglyoxal (MG)
    → MG is produced at 5-10× higher rate in T1DM vs. normoglycemia
    → MG → AGE formation (MG-derived AGEs: CEL, MG-H1)
    
MG → TRPA1 (Andersson 2013 PNAS 110(23):9399-9404):
    MG → binds Arg/Lys sites near TRPA1 Cys gating region → induces channel opening
    → MG-activated TRPA1 drives diabetic peripheral neuropathy pain
    
In T1DM rosacea context:
    Systemic MG elevation → TRPA1 on facial sensory neurons sensitized/pre-activated
    → Lower threshold for TRPA1 activation by 4-HNE (Loop 4) or food triggers
    → T1DM patients have LOWER THRESHOLD for rosacea flare triggers than non-diabetic rosacea
    → Explains clinical observation: T1DM rosacea patients often more reactive to food/environmental triggers
    
MG and TRPV1 (run_015) cross-talk:
    MG → TRPA1 → sensitizes TRPV1 → double neurogenic activation from same MG
    → MG-TRPA1-TRPV1 sensitization loop: T1DM uniquely vulnerable
```

## Food Triggers: TRPA1 Is the Molecular Mechanism

**Classic rosacea food triggers — TRPA1 agonist chemistry:**
```
Trigger → Active compound → TRPA1 agonism → Flushing

Garlic → allicin (diallyl disulfide; electrophilic sulfur compound):
    Allicin + Cys621 → disulfide adduct → TRPA1 gating
    Evidence: Bautista 2005 Cell 123(6):1013-1021: allicin → TRPA1 confirmed

Alcohol → acetaldehyde (ethanol oxidation product):
    Acetaldehyde → Michael addition to TRPA1 Cys → activation
    + Ethanol itself weakly activates TRPA1 (lower potency than acetaldehyde)
    Evidence: Emir 2012 Anesthesiology: TRPA1 mediates alcohol-evoked pain (same channel, rosacea)

Cinnamon → cinnamaldehyde (α,β-unsaturated aldehyde):
    One of the STRONGEST known TRPA1 agonists (EC50 ~10 µM)
    Cinnamaldehyde → Cys641/665 modification → rapid, prolonged TRPA1 activation
    Evidence: Bandell 2004 Neuron 41(6):849-857: cinnamaldehyde → TRPA1 founding paper

Hot food, spices (wasabi, mustard) → allyl isothiocyanate (AITC):
    AITC → primary TRPA1 agonist from cruciferous plant defense compounds
    Note: sulforaphane (from broccoli sprouts) is a mild TRPA1 agonist (isothiocyanate structure)
    → TRPA1 side note: oral sulforaphane → slight TRPA1 activation → may transiently worsen 
      neurogenic flushing initially before anti-inflammatory effects dominate

Vinegar → acetic acid (weak acid + TRPA1 agonist):
    Acetic acid at low pH → TRPA1 gating (pH-sensitive component)
    
Fermented foods → putrescine + histamine → mast cell activation (different pathway; existing framework)
    BUT also → short-chain aldehydes → TRPA1 → neurogenic co-activation
```

## Protocol Coverage and Gaps

**Upstream TRPA1 agonist reduction (existing protocol):**
```
AzA (run_082, mechanism 4): 4-HNE ↓ → less TRPA1 agonist [most direct]
Sulforaphane → Nrf2 → HO-1/NQO1 → less lipid peroxidation → less 4-HNE/H₂O₂
SIRT3 → SOD2 → O₂•⁻ → H₂O₂ ↓ (run_090) → less TRPA1 redox activation
RAAS control (run_092): Ang II ↓ → Nox2 ↓ → H₂O₂ ↓ → less TRPA1 redox activation
CAPE/antioxidants → H₂O₂/4-HNE ↓
Glycemic control (T1DM primary): MG production ↓ → less T1DM-specific TRPA1 activation
```

**No direct TRPA1 inhibitor in protocol:**
```
TRPA1 antagonists (research only):
    HC-030031 (TRPA1 antagonist): research compound; no clinical use
    A-967079: TRPA1 antagonist; clinical trials for pain; not rosacea-specific
    
Clinically available TRPA1 modulation:
    Ketamine (TRPA1 block at anesthetic doses): not appropriate for rosacea
    Menthol (at moderate concentrations): counter-activates TRPA1 at low temp/cold → complex
    
Practical TRPA1 management: upstream agonist reduction (existing protocol) + lifestyle avoidance
    of dietary TRPA1 agonists (garlic, alcohol, cinnamon, hot spices) during flares
```

**Sulforaphane TRPA1 caveat:**
```
Sulforaphane (broccoli sprouts; run_014 second NF-κB suppressor + Nrf2):
    Isothiocyanate structure → mild TRPA1 activation
    → Initial TRPA1 agonism may cause brief vasodilation/flushing on first use
    → At low doses (morning broccoli sprout portion): TRPA1 activation brief, Nrf2 benefit sustained
    → Clinical: take sulforaphane with food; start low dose; some patients notice initial flush
      (TRPA1-mediated) that resolves as Nrf2 anti-inflammatory effect establishes (2-4 weeks)
```

## Kill Criteria

**Kill A: TRPA1 Expression in Facial Dermis Sensory Neurons Not Definitively Quantified**
Buhl 2017 and Mascarenhas 2019 show TRPA1 in rosacea tissue but did not quantify relative to non-inflamed facial skin, and did not directly link TRPA1 to rosacea severity.
**Status:** Real limitation. The mechanistic case is strong (TRPA1 is the primary electrophile sensor; 4-HNE activates TRPA1 at concentrations found in rosacea skin). The clinical evidence is indirect (food trigger correlation with TRPA1 agonism). The Buhl/Mascarenhas data confirms expression but not causation. Framework position: mechanistically high-confidence; causation in rosacea clinically plausible but not RCT-confirmed.

**Kill B: Upstream Agonist Reduction (4-HNE ↓) Already Covers TRPA1 Indirectly**
AzA (mechanism 4), sulforaphane, and antioxidants reduce 4-HNE production → TRPA1 receives fewer agonists. TRPA1 analysis is mechanistic context for existing protocol rather than a new target.
**Status:** Not killed as a conceptual run. TRPA1 adds: (1) the Loop 4 → M8 direct bridge (previously unmapped); (2) food trigger molecular explanation (garlic/alcohol/cinnamon = TRPA1 agonists); (3) T1DM MG → TRPA1 specific mechanism; (4) CGRP → MRGPRX2 → tryptase → PAR-2 → KLK5 cross-loop connection; (5) sulforaphane TRPA1 caveat (clinically relevant for protocol management). The value is the INTEGRATION, not a new agent.
