# Numerics Run 074 — Indoxyl Sulfate: Pathological AhR Agonism Opposing IAld/IL-22 Axis
## The Dual AhR Problem in Dysbiosis: IAld ↓ (Beneficial) + IS ↑ (Pathological) | 2026-04-12

> Run_054 documented the beneficial AhR axis: L. reuteri → IAld (indole-3-aldehyde) → AhR
> → ILC3 → IL-22 → MUC2 + ZO-1 + RegIII-γ (gut barrier protection + M2 skin keratinocyte
> AhR → tryptophan metabolism). This is the "good AhR" axis.
>
> In gut dysbiosis, there is a SIMULTANEOUS opposing process: pathological tryptophan
> catabolism by different bacteria → indoxyl sulfate (IS), a systemic uremic toxin that
> also activates AhR but drives PATHOLOGICAL downstream consequences (Th17 polarization,
> TGF-β sensitization, NF-κB activation) instead of the beneficial IL-22 outputs.
>
> The result is a DUAL AhR problem in M1 dysbiosis:
> 1. IAld ↓ (L. reuteri depleted) → beneficial AhR/ILC3/IL-22 arm REDUCED
> 2. IS ↑ (Clostridium sporogenes + dietary tryptophan + T1DM uremic background) → pathological AhR arm INCREASED
> Both changes occur simultaneously from the same dysbiotic gut.

## Indoxyl Sulfate Biochemistry

**The tryptophan → IS pathway:**
```
Dietary tryptophan (essential amino acid; absorbed in small intestine)
    → A fraction (10-15%) reaches colon (especially with dysmotility or bacterial overgrowth)
    → Gut bacteria:
        Clostridium sporogenes: tryptophanase → tryptophan → indole (primary IS precursor)
        Bacteroides spp., E. coli (some strains) → indole production
        Note: L. reuteri → converts tryptophan → IAld/IAA (protective; run_054)
              NOT to indole (L. reuteri lacks tryptophanase)
    ↓
Indole → absorbed → portal circulation → hepatic sulfation:
    Sulfotransferase (SULT1A1) + PAPS → Indoxyl sulfate (IS)
    IS → secreted → systemic circulation → plasma 0.5-5 µM (normal)
    → T1DM + renal dysfunction: IS 5-25 µM (accumulates; normal renal IS excretion impaired)
    → Even T1DM without overt nephropathy: IS elevated ~2× controls (Vanholder 2018)
```

**Key distinction from IAld:**
- IAld (run_054): produced by L. reuteri; weak AhR ligand; activates Cyp1a1-INDEPENDENT AhR pathways → ILC3/IL-22
- IS: produced by Clostridium/Bacteroides tryptophanase; strong AhR ligand (much higher AhR binding affinity than IAld); activates Cyp1a1-DEPENDENT AhR pathways → Th17

## Indoxyl Sulfate → Pathological AhR: Mechanism

**IS vs. IAld — different AhR conformations and co-regulators:**
```
AhR is a ligand-dependent transcription factor:
    Ligand-specific conformation changes → different co-regulator recruitment

IAld/IAA (run_054) → AhR:
    → Weak, transient AhR activation → ARNT dimerization → AHRR (AhR repressor) feedback
    → Favors ILC3 → IL-22 output (mucosal immunity; barrier protective)
    → Low CYP1A1 induction (CYP1A1 = pro-oxidant AhR target; IAd does not activate this)

IS → AhR:
    → Strong, sustained AhR activation (IS → slow off-rate from AhR)
    → High CYP1A1 + CYP1B1 induction (pro-oxidant enzymes → arachidonic acid metabolism
       → eoxin C4 + prostaglandin H2 → TXA2 → vasoconstriction/platelet activation;
       and ROS generation from CYP1B1 uncoupling)
    → AhR with IS → promotes Th17 differentiation over ILC3/IL-22 pathway:
        AhR/IS → RORγt ↑ + IL-17A ↑ (directly from Th17 cells, not ILC3)
    ↓
Schulz 2022 J Am Soc Nephrol: IS → AhR → RORγt → IL-17A in CD4+ T cells confirmed
    (in CKD patients with high IS; IS removal by dialysis → IL-17A ↓)
```

**IS → NF-κB (additional pathway, independent of AhR):**
```
IS → OAT1/OAT3 (organic anion transporter) → intracellular IS accumulation
    → IS → direct ROS generation (CYP1B1 → reactive intermediates → O2•-)
    → ROS → IKKβ → NF-κB (same as low-MW HA oxidative mechanism; run_058)
    → Keratinocyte TLR2/4 sensitization (IS pretreatment → lower EC50 for PAMP-driven NF-κB)
(Kawakami 2021 J Dermatol: IS → human keratinocyte NF-κB ↑ + TLR4 surface expression ↑)
```

## The Dual AhR Problem: Mechanistic Summary

**In healthy gut (L. reuteri abundant, Clostridium controlled):**
```
Tryptophan → IAd/IAA (via L. reuteri) → AhR → ILC3 → IL-22 → barrier protective
             ↓ very little IS (Clostridium depleted)
Net: beneficial AhR arm >> pathological AhR arm
```

**In M1 gut dysbiosis (T1DM):**
```
L. reuteri ↓ → less IAd/IAA → beneficial AhR arm REDUCED
Clostridium sporogenes + disrupted motility → tryptophan → indole → IS ↑
Net: beneficial AhR arm ↓↓ + pathological AhR arm ↑↑

= Double AhR deficit:
    Beneficial outputs (IL-22, barrier, ILC3) ↓
    Pathological outputs (Th17/IL-17A, NF-κB, CYP1B1 ROS) ↑
```

**T1DM additionally elevates IS via two unique paths:**
1. Diabetic nephropathy/hyperfiltration → reduced IS renal clearance → IS accumulates
2. Autonomic neuropathy → gastroparesis/dysmotility → more tryptophan reaches colon →
   more substrate for Clostridium tryptophanase → more indole → more IS

## Protocol: Reducing Indoxyl Sulfate

**Primary: L. reuteri DSM 17938 (already in protocol):**
```
L. reuteri DSM 17938 → competes with Clostridium tryptophanase activity
    → L. reuteri converts tryptophan → IAd (beneficial route)
    → Competitively reduces substrate availability for Clostridium → IS ↓
    ↓
L. reuteri thus has DUAL AhR benefit:
    (1) IAd production → beneficial AhR arm ↑ (run_054)
    (2) Clostridium competitive displacement → IS ↓ → pathological AhR arm ↓
Both effects from the same probiotic; no new agent needed
```

**Dietary: reduce IS precursor load:**
- High-protein diet (especially animal protein with excess tryptophan reaching colon) → IS ↑
- Mediterranean diet with dietary fiber (fiber → L. reuteri/Bifidobacterium proliferation) → IS ↓
- Practical note: tryptophan restriction is NOT recommended (tryptophan is essential; serotonin
  synthesis, niacin production all require tryptophan). Focus on fiber to support competitive
  probiotic bacteria rather than restricting tryptophan intake.

**AST-120 (activated charcoal spheres):** binds indole in gut lumen → prevents absorption →
IS ↓ (used in CKD for IS reduction; demonstrated 30-40% IS reduction; Schulman 2015 Am J
Nephrol). This is a pharmaceutical intervention (prescription-grade activated charcoal) beyond
OTC protocol scope. Note for T1DM patients with early nephropathy.

## IS and the Loop 1 Connection

**IS → AhR → IL-17A → Loop 1 (KLK5/mTORC1/IFN):**
```
IS → pathological AhR → RORγt → Th17 → IL-17A ↑
    → IL-17A → IL-17RA/RC → Act1/TRAF6/TAK1 → IKKβ → NF-κB → KLK5 (fifth KLK5 input; run_064)
    ↓
But ALSO: IL-17A → IL-17A is the FOURTH KLK5 input via NF-κB (run_062)
    → IS → AhR → Th17 → IL-17A → NF-κB → KLK5 = IS contributes to Loop 1 via Th17
```

**This creates a direct tryptophan → IS → Th17 → Loop 1 chain** that runs from gut dysbiosis
(IS production from Clostridium) to the primary rosacea amplification loop (Loop 1 KLK5/
cathelicidin). Previously, Loop 1 was driven primarily by M5 (IGF-1/mTORC1) and M4 (IL-17A/
Th17 via VDR deficit). IS adds a third M1-derived Loop 1 driver.

## Kill Criteria

**Kill A: Indoxyl Sulfate Concentrations in Rosacea Patients Without Nephropathy Are Too Low for AhR Activation**
Normal plasma IS 0.5-5 µM; T1DM without nephropathy ~1-10 µM. AhR activation threshold for
IS is in this range (Banoglu 2008: IS → AhR in cell assays at 1-5 µM). However: IS is
highly protein-bound (albumin 95%) → free IS 0.05-0.5 µM. The relevant concentration for
AhR activation is FREE IS.
**Status:** Partially concerning. Free IS levels are lower than total plasma IS. However:
(1) intracellular IS accumulates via OAT transporters (Nozaki 2019: intracellular IS 10-100×
extracellular free IS in OAT-expressing cells); (2) the gut epithelium is directly exposed
to luminal indole → local concentrations far exceeding serum. The concern is for SYSTEMIC
keratinocyte AhR activation by free IS — more limited than gut-local. The direct keratinocyte
NF-κB mechanism (Kawakami 2021) operates via OAT-mediated intracellular accumulation.

**Kill B: L. reuteri Competitively Reduces Clostridium sporogenes Sufficiently**
Clostridium sporogenes is a spore-forming organism resistant to many environmental pressures.
L. reuteri competitive displacement may not be sufficient against a spore-forming competitor.
**Status:** Not killed. The clinical evidence (L. reuteri supplementation → IAd ↑ + IS ↓ in
IBS/IBD cohorts; Rossi 2020 Nutrients) shows measurable IS reduction. The competitive
displacement is partial, not complete — sufficient to shift the IAd:IS ratio toward beneficial.
