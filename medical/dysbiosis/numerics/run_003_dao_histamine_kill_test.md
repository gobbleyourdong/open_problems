## Hypothesis

**Gut-derived histamine → DAO deficiency → systemic histamine exposure → mast cell sensitization → rosacea flushing**

**Kill criterion:** DAO activity assay + dietary challenge. If DAO normal AND no flushing improvement on low-histamine diet → pathway not active → skip this intervention.

## Mechanistic Chain

```
Histamine-producing gut bacteria
(Morganella morganii, Klebsiella pneumoniae,
 some Lactobacillus spp., Hafnia alvei)
     ↓
Histamine synthesized from dietary histidine
     ↓
Normal: DAO (diamine oxidase) degrades histamine in gut wall
        HNMT (histamine N-methyltransferase) handles remainder
        → Only trace histamine enters portal circulation
     ↓
DAO DEFICIENCY: inadequate degradation
        → Elevated histamine in portal blood
        → Hepatic partial clearance
        → Elevated systemic histamine
     ↓
Mast cell H1/H4 receptor activation
     ↓
Vasodilation → FLUSHING (rosacea subtype 1)
Neurogenic inflammation → burning sensation
Prostaglandin release → sustained vasodilation
```

## DAO Activity Assay

**What it measures:** Serum DAO enzymatic activity. Units: HDU/mL (histamine degrading units per mL).

**Reference range (varies by lab):**
- >10 HDU/mL: sufficient activity
- 3-10 HDU/mL: marginal deficiency
- <3 HDU/mL: significant deficiency (associated with histamine intolerance symptoms)

**Lab availability:**
- **US:** GENOVATION DIAGNOSTICS (DAO Test), Cleveland HeartLab, some specialty panels
- **Germany:** Medivere (market leader for DAO testing in Europe)
- **UK:** Microtrace Minerals, York Test
- **Cost:** ~$100-200 (US); €50-150 (Germany)

**Alternative marker: 24-hour urinary methylhistamine**
- More stable than plasma histamine
- Reflects overall histamine turnover over 24 hours
- Normal: <9 µg/mg creatinine
- Elevated: indicates increased histamine load (production OR reduced degradation)
- Easier to collect than serum DAO but less direct

**Which to order:** DAO activity (serum) is the standard. If unavailable, 24h urine methylhistamine.

## Dietary Challenge Protocol

### Phase 1: Baseline (2 weeks, normal diet)
- Keep a flushing diary: frequency (episodes/day), severity (1-10), duration (minutes), triggers
- Also track: DLQI (Dermatology Life Quality Index) — validated questionnaire
- No dietary changes

### Phase 2: Low-Histamine Diet (4 weeks)
**ELIMINATE:**
| High-histamine foods | High-histamine drinks |
|---------------------|----------------------|
| Fermented cheese (aged cheddar, parmesan, blue) | Wine (all types — especially red) |
| Fermented meats (salami, pepperoni, prosciutto) | Beer |
| Canned/smoked fish | Champagne, sake |
| Fermented vegetables (sauerkraut, kimchi) | Kombucha |
| Soy sauce, fish sauce, worcestershire | Vinegar-based drinks |
| Long-cooked tomato sauces | |
| Avocado | |
| Eggplant, spinach (high free histamine) | |
| Fermented condiments | |

**ALSO ELIMINATE (histamine liberators — cause mast cell degranulation without containing histamine):**
- Alcohol (direct mast cell activator)
- Strawberries, citrus, tomatoes
- Shellfish (esp. shrimp)
- Egg white
- Chocolate

**KEEP (low-histamine):**
- Fresh meat and fish (frozen is acceptable; avoid canned)
- Fresh vegetables (not the above)
- Rice, potato, sweet potato
- Most fruits except citrus and strawberry
- Coconut milk, hemp milk (avoid cow dairy if also casein-sensitive)

### Phase 3: DAO Supplementation (4 weeks, still on low-histamine diet)
- DAOSIN (porcine kidney DAO extract, 10,000 HDU/capsule) — take 15-30 min before meals
- Alternatively: Histame, ImmuHist, Seeking Health HistaminX
- Same flushing diary

### Phase 4: Reintroduction
- Reintroduce one high-histamine food/week
- Record which foods trigger flushing
- Identify individual threshold

## Kill Criteria

| Result | Interpretation | Protocol Action |
|--------|---------------|-----------------|
| Low DAO + diet improves flushing >50% | Histamine axis CONFIRMED | Add DAO supplementation + low-histamine diet to protocol. Prioritize Lactobacillus rhamnosus (doesn't produce histamine, may displace histamine-producers). |
| Low DAO + diet doesn't help | DAO deficiency present but histamine not the main flushing driver | Look elsewhere: vascular (niacinamide, carvedilol low dose for flushing), TRP channels (TRPV1 — capsaicin avoidance), neurogenic |
| Normal DAO + diet improves flushing | Histamine effect via histamine-liberators or other mechanism, not DAO deficiency | Keep diet modification; skip DAO supplement |
| Normal DAO + no diet change | Histamine axis NOT active | Pathway excluded. Move to other flushing mechanisms. |

## Other Flushing Mechanisms (if histamine axis excluded)

For completeness — if the kill test is negative, what else?

1. **KLK5/LL-37/TRPV1 pathway** (rosacea-specific): LL-37 → TRPV1 activation on sensory nerves → neuropeptide release (CGRP, substance P) → vasodilation. NMF (National Rosacea Society) has data. Targets: topical brimonidine (vasoconstriction), oxymetazoline, ketotifen (mast cell stabilizer).

2. **Th17/IL-17 pathway**: from Mountain 7 (oral P. gingivalis) or gut dysbiosis → Th17 skew → IL-17 → skin vasodilation. Anti-IL-17 (secukinumab, ixekizumab) used in psoriasis; not tested for rosacea specifically.

3. **Demodex/serine protease cascade**: Demodex antigen → KLK5 upregulation → cathelicidin processing → LL-37 → inflammatory and vascular response. Treatment: ivermectin (Soolantra) — has anti-flushing effect beyond antimicrobial action (direct TRPV1 modulation).

4. **Sympathetic dysregulation**: rosacea flushing can be sympathetic-mediated. Carvedilol low dose, clonidine have some evidence for flushing reduction.

## Low-Histamine Probiotics

If histamine axis is active, avoid:
- Lactobacillus casei
- Lactobacillus delbrueckii
- Lactobacillus bulgaricus
- Lactobacillus helveticus
- (all documented histamine producers)

Use instead:
- Lactobacillus rhamnosus GG (HistaminX formulations)
- Bifidobacterium longum (histamine-degrading)
- Lactobacillus plantarum (histamine-degrading, produces DAO-like activity)
- Lactobacillus salivarius
