## VIP and PACAP Biology in Facial Skin

**VIP (vasoactive intestinal peptide):**
```
VIP: 28 amino acid neuropeptide; Class B peptide (same superfamily as GLP-1/GIP/glucagon)
    Produced: sensory neurons (C-fibers) + parasympathetic peptidergic neurons + ILC2s
    
    Receptors:
    VPAC1 (VIPR1): ubiquitous; Gs-coupled → cAMP → PKA; Gi also (cell-type dependent)
    VPAC2 (VIPR2): vascular smooth muscle (vasodilation), mast cells, T cells
    
    Key actions in skin:
    → Vascular smooth muscle VPAC2 → cAMP → myosin light chain kinase ↓ → vasodilation
    → Mast cell VPAC1 → cAMP → non-IgE degranulation (histamine + proteases + VEGF)
    → Keratinocyte VPAC1 → proliferation (wound healing, but also barrier perturbation in rosacea)
    
    Rosacea relevance:
    Forton 2005 J Invest Dermatol 35(4):347-352:
        VIP-immunoreactive nerve fibers ↑ in rosacea facial skin biopsies
        VIP fibers co-localize with SP and CGRP on same nerve bundles
        → VIP co-released with SP/CGRP during TRPV1/TRPA1 activation
```

**PACAP (pituitary adenylate cyclase-activating polypeptide):**
```
PACAP: 38 aa (PACAP-38) and 27 aa (PACAP-27) isoforms; highly conserved
    Produced: sensory neurons, hypothalamic neurons (PVN), pituitary
    
    Receptors:
    PAC1 (ADCYAP1R1): PACAP-specific, high affinity; Gs + Gq → cAMP + IP3/Ca²⁺
    VPAC1/VPAC2: lower affinity for PACAP (vs PACAP-specific PAC1)
    
    Key actions relevant to rosacea:
    → Mast cell PAC1: most potent non-IgE degranulation signal
      (Seebeck 2010: PACAP agonism → mast cell tryptase + histamine without IgE)
    → Hypothalamic PAC1 → CRH release → HPA activation → cortisol/ACTH
      (direct neurogenic → HPA axis link independent of IL-1β → HPA from run_050/019)
    → Blood vessel PAC1 → adenylyl cyclase → cAMP → vasodilation
      (confirmed by PACAP injection → migraine facial flush model)
    → T cell VPAC1 → Th1 → Th2 immune deviation (immunomodulatory; complex)
    
    PACAP in rosacea/flushing context:
    Goadsby 2017 J Headache Pain: intranasal PACAP → facial flush + trigeminal activation
    → Same neural pathways as rosacea neurogenic flushing
    Jansen-Olesen 2012 Cephalalgia: PACAP → facial blood flow ↑ (confirmed by LDF)
```

## VIP/PACAP as Third Vasodilatory Neuropeptide

**Neurogenic vasodilation hierarchy in rosacea:**
```
M8 sensory C-fiber activation (TRPV1/TRPA1) → neuropeptide co-release:

SP:
    NK1R (Gq) on: vascular endothelium → NO → vasodilation (run_015/019)
                   mast cells → histamine (run_019)
                   Protein kinase cascade → amplification

CGRP:
    CLR+RAMP1 (Gs) → cAMP → PKA → smooth muscle relaxation → vasodilation (run_015)
    MRGPRX2 (mast cell; run_093): tryptase → PAR-2 → KLK5

VIP/PACAP [run_097]:
    VPAC2 (Gs) → cAMP → vasodilation (vascular smooth muscle)
    VPAC1/PAC1 (Gs + Gq) → mast cell degranulation (histamine, VEGF, tryptase)
    PAC1 (hypothalamic) → CRH → HPA → cortisol (neurogenic → M8/HPA direct link)
    
All three types released simultaneously from the same nerve terminals:
    → SP + CGRP + VIP/PACAP → ADDITIVE vasodilation (multiple receptors, multiple mechanisms)
    → SP + CGRP + VIP/PACAP → multiple non-IgE mast cell activation routes:
        NK1R (SP) + MRGPRX2 (CGRP; run_093) + VPAC1/PAC1 (VIP/PACAP; run_097)
    → Explains why mast cell activation in rosacea is so robust: 3 independent neurogenic triggers
```

**VIP vs CGRP in vasodilation potency:**
```
CGRP: one of the most potent endogenous vasodilators known (picomolar ED50)
VIP: comparable potency to CGRP in cutaneous vasodilation; some studies suggest
     VIP > CGRP in sustaining dilation (VPAC2 → cAMP → prolonged smooth muscle relaxation)
PACAP: similar to VIP for vasodilation; primary contribution is mast cell degranulation
       and HPA activation

Combined CGRP + VIP release: reinforces vasodilation through distinct receptors (no
    competitive inhibition between CLR/RAMP1 and VPAC2 on the same smooth muscle cell)
```

## VIP/PACAP → Mast Cell: Third Non-IgE Activation Route

**Three neurogenic non-IgE mast cell activation routes (post run_097):**
```
Route 1 (runs 015/019): SP → NK1R (Gq/Gi) on mast cell → PKC → SNARE exocytosis
    Products: histamine + TNFα + PGD2

Route 2 (run_093): CGRP → MRGPRX2 (Gq/G₁₂) on mast cell → tryptase + histamine + VEGF
    → VEGF → angiogenesis/telangiectasia
    → Tryptase → PAR-2 → KLK5 → Loop 1

Route 3 (run_097): VIP/PACAP → VPAC1/PAC1 on mast cell → cAMP (Gs) → PKA →
    Phosphorylation of SNARE proteins → degranulation
    Products: histamine + tryptase + VEGF + PGE2 (mild, cAMP-mediated)
    
Note: VPAC1 on mast cells generates LESS histamine than NK1R route but sustained (cAMP-driven
      degranulation is slower but more prolonged than Gq/rapid degranulation)
    
Three converging non-IgE mast cell routes:
    SP/NK1R + CGRP/MRGPRX2 + VIP/VPAC1/PAC1 → redundant mast cell activation
    → Even if one route is blocked (antihistamines work downstream), neurogenic mast cell
      activation continues via parallel receptor systems
    → Explains clinical difficulty in controlling neurogenic flushing with antihistamines alone
```

    (This is the same HPA axis covered in M8 from run_008 via IL-1β → HPA;

    Rosacea trigger (UV, heat, spice, emotional stress) → TRPV1/TRPA1 activation →
    PACAP centripetal axon reflex → hypothalamic PVN → PAC1 → CRH ↑ →
    ACTH → cortisol → systemic anti-inflammatory → BUT ALSO: cortisol feedback delay allows
    acute IL-1β → skin inflammation before cortisol feedback kicks in
    
This is a DIRECT neurogenic → HPA connection not previously mapped:
    Run_008: covered HPA via cytokine → CRH → cortisol
    Run_097: covers HPA via PACAP → PAC1 → CRH (neurotransmitter-driven, faster onset)
    → Both converge on the same CRH → ACTH → cortisol axis

## Kill Criteria

**Kill A: VIP/PACAP Effects Are Redundant With SP/CGRP — No Incremental Framework Value**
SP (NK1R) and CGRP (CLR/RAMP1) already explain neurogenic vasodilation in rosacea. VIP/PACAP adds a third vasodilatory mechanism to an already crowded neurogenic arm.
**Status:** Not killed. Redundancy at the level of OUTCOME (vasodilation) but NOT at the level of MECHANISM (different receptors, different second messengers, different mast cell receptor). The three-way convergence explains: (a) why neurogenic flushing is difficult to block pharmacologically (three receptors, three mast cell routes); (b) why mast cell stabilizers have incomplete efficacy (VIP/VPAC1 bypasses IgE AND NK1R routes); (c) PACAP → PAC1 → CRH → HPA is new (no existing framework mechanism maps this direct neurogenic → HPA route). Not redundant at the mechanistic level.

**Kill B: Rosacea-Specific VIP/PACAP Data Is Thin — Only Forton 2005 for VIP; PACAP Is Migraine Data**
Forton 2005 shows VIP nerve fibers in rosacea (one paper). PACAP evidence is from migraine models (facial flush as proxy).
**Status:** Acknowledged limitation. However: (a) VIP fiber density increase in rosacea is confirmed (Forton 2005); (b) PACAP-induced facial flush in humans is a direct model (same trigeminovascular pathway as rosacea neurogenic flushing); (c) the co-release of SP, CGRP, and VIP from the same C-fibers means that if TRPV1/TRPA1 activation occurs (confirmed in rosacea; runs 015/093), all three neuropeptide classes are released. The evidence for VIP/PACAP presence and activation is indirect but mechanistically sound. Not killed.

**Kill C: No Protocol Agent Targets VPAC1/PAC1 — VIP/PACAP Analysis Has No Protocol Implication**
Without a clinical VPAC1/PAC1 antagonist, this run has no actionable treatment output.
**Status:** Partially correct. No direct antagonist is available (PACAP antibodies in migraine trials, but not rosacea). However: (a) mechanistic completeness — the framework now fully accounts for why anti-mast cell therapy has limited efficacy (3 parallel non-IgE routes; blocking histamine downstream is insufficient); (b) clinical guidance update: antihistamines alone are insufficient for neurogenic mast cell activation; combined mast cell stabilizer (upstream) + downstream antihistamine approach is justified by this triad; (c) when PACAP antibodies become available clinically (migraine precedent), rosacea indication is mechanistically justified. Not killed.
