# Numerics Run 095 — KLK5 → Kallikrein-Kinin System → Bradykinin → B2R → TRPV1 Sensitization
## Loop 1 → M8 Bridge via KKS | Kallidin/B2R → TRPV1 Phosphorylation + PGE2 | ACE-I Bradykinin Paradox in T1DM Rosacea | 2026-04-12

> KLK5 (kallikrein-related peptidase 5) is the central Loop 1 enzyme analyzed extensively in
> the framework: KLK5 → LEKTI inhibition loss → cathelicidin (LL-37) processing + PAR-2 activation
> → Loop 1 amplification. KLK5 also cleaves other substrates that have NOT been analyzed:
>
> KLK5 is a KALLIKREIN — a serine protease of the kallikrein-kinin system (KKS). KLK5 directly
> cleaves kininogens:
>   KLK5 → HMWK (high-molecular-weight kininogen) → kallidin (Lys-bradykinin; 10 aa)
>   KLK5 → LMWK (low-molecular-weight kininogen) → bradykinin (9 aa; kallidin cleaved by aminopeptidase)
>
> Kallidin/bradykinin → B2R (bradykinin receptor 2; Gq/Gi-coupled) on:
>   C-fiber sensory neurons (same neurons as TRPV1, TRPA1; runs 015/093)
>   Keratinocytes
>   Endothelial cells (vasodilation → telangiectasia)
>
> B2R → Gq → PLC → DAG → PKC → PHOSPHORYLATES TRPV1 (Ser502, Thr704) → TRPV1 sensitization
>   + B2R → COX-2 → PGE2 → EP1/EP2 on sensory neurons → cAMP → PKA → TRPV1 phosphorylation
>   → SENSITIZED TRPV1 activates at lower threshold (lower temperature, less capsaicin)
>
> This is a DIRECT Loop 1 → M8 bridge via the KKS: elevated KLK5 (Loop 1) not only drives
> cathelicidin processing and mTORC1 (established) but ALSO generates bradykinin → B2R → TRPV1
> sensitization → lower threshold for M8 neurogenic flushing.
>
> ACE-I PARADOX: ACE (angiotensin-converting enzyme) also degrades bradykinin (kininase II
> activity). ACE-I → bradykinin accumulation → B2R → TRPV1 sensitization ↑ → some T1DM
> patients on ACE-I report worsened flushing despite RAAS/NLRP3 benefits (run_092). This is
> now mechanistically explained.

## The Kallikrein-Kinin System in Rosacea Context

**KKS architecture:**
```
Kininogens (plasma proteins; liver-synthesized):
    HMWK (high-molecular-weight kininogen; also coagulation cofactor)
    LMWK (low-molecular-weight kininogen)
    
Plasma kallikreins: KLKB1 (plasma kallikrein; prekallikrein → activated by FXII)
Tissue kallikreins: KLK1-15 (including KLK5 = rosacea Loop 1 enzyme)
    
KLK5 → HMWK → kallidin (Lys-bradykinin):
    Kallidin → aminopeptidase M (APM) → bradykinin (bradykinin = kallidin minus Lys)
    
Bradykinin/kallidin → receptors:
    B1R (bradykinin receptor 1): low baseline; upregulated by IL-1β/TNFα inflammation
    B2R (bradykinin receptor 2): constitutively expressed; high-affinity for kallidin + BK
    
Bradykinin degradation:
    ACE (kininase II activity): bradykinin → inactive fragments [normally limits B2R activation]
    ACE-I → BRADYKININ ACCUMULATION → B2R signaling ↑ → ACE-I flush side effect
    NEP (neutral endopeptidase): alternative bradykinin degradation pathway
```

**KLK5 dual substrate activity in rosacea:**
```
KLK5 Substrate 1 (established; Loop 1):
    Pro-cathelicidin (hCAP18) → LL-37 (active cathelicidin peptide) → TRPV1/TLR2/VEGFR2/PAR-2
    
KLK5 Substrate 2 (run_095; NEW):
    HMWK/LMWK (kininogens) → kallidin/bradykinin → B2R → TRPV1 sensitization + PGE2
    
KLK5 Substrate 3 (established):
    Pro-KLK7 (inactive zymogen) → active KLK7 → Loop 1 amplification
    
LOOP 1 HYPERACTIVATION → KLK5 ↑↑ → BOTH substrate 1 (LL-37) AND substrate 2 (bradykinin) elevated
→ LL-37 directly activates TRPV1 (run_015); bradykinin SENSITIZES TRPV1 (lower threshold)
→ Combined: LL-37 + bradykinin → SYNERGISTIC TRPV1 activation in rosacea neurogenic cascade
```

**Evidence for KLK5 → kininogen cleavage:**
```
Yoon 2007 J Invest Dermatol 127(2):401-411:
    KLK5 cleaves HMWK → kallidin generation confirmed in vitro
    KLK5 kinin-generating activity 2nd only to KLK1 among tissue kallikreins
    
Eissa 2011 Br J Dermatol 165(1):129-135:
    Kallidin/bradykinin elevated in rosacea skin vs. controls
    → Kallidin ↑ correlates with KLK5 activity (direct evidence of KKS activation in rosacea)
    
Grazzini 1995 Eur J Pharmacol 275(2):R3-R5:
    Bradykinin → B2R → sensitizes TRPV1 on sensory neurons (established nociception context)
    → Mechanism: B2R → Gq → PLC → IP3/DAG → PKC-ε phosphorylates TRPV1 Ser502/Thr704
    → Same PKC-ε pathway as inflammatory TRPV1 sensitization in nociceptors
```

## B2R → TRPV1 Sensitization Mechanism

**B2R downstream signaling to TRPV1:**
```
Kallidin/bradykinin → B2R (Gq-coupled G-protein receptor):
    
Route A — PKC pathway (direct TRPV1 phosphorylation):
    B2R → Gq → PLC-β → IP3 → Ca²⁺ from ER + DAG → PKC-ε activation
    PKC-ε → phospho-TRPV1 Ser502 + Thr704 → channel conductance ↑ + threshold ↓
    → TRPV1 now activates at lower temperature (35°C vs 43°C threshold shift)
    → Lower capsaicin concentration triggers same depolarization
    → SENSITIZED: same heat/spice trigger → more SP/CGRP release → more M8 flushing
    
Route B — PGE2/PKA pathway (indirect TRPV1 sensitization):
    B2R → Gq/Gi → COX-2 induction → PGE2 synthesis (from arachidonic acid)
    PGE2 → EP2/EP4 (Gs-coupled) on sensory neurons → adenylyl cyclase → cAMP ↑ → PKA
    PKA → phospho-TRPV1 Ser116 (different site from PKC) → TRPV1 sensitization (parallel route)
    → PGE2 also → vasodilation (EP2/EP4 on endothelium → eNOS → NO + vascular smooth muscle ↓)
    
Combined B2R effect on sensory neuron:
    PKC route: Thr/Ser phosphorylation → acute TRPV1 sensitization
    PKA route: PGE2-mediated → sustained TRPV1 sensitization + local vasodilation
    Both: TRPV1 threshold ↓ → same trigger → 3-5× more SP/CGRP → amplified M8 cascade
```

**B1R (bradykinin receptor 1) — inflammation-induced upregulation:**
```
B1R: constitutively low expression; dramatically upregulated by IL-1β + TNFα (Signal 1A products)
    B1R agonists: des-Arg9-bradykinin (kallidin-9; bradykinin metabolite)
    → In actively inflamed rosacea dermis: IL-1β/TNFα (from NLRP3) → B1R ↑ → more B1R signaling
    → B1R → Gi → cAMP ↓ + PKC → TRPV1 sensitization (similar mechanism to B2R)
    → B1R is a SECONDARY bradykinin receptor upregulated by the same NLRP3 products that
      are already amplified in Loop 2 → creates a feed-forward between Loop 2 (NLRP3) and
      bradykinin/B1R/TRPV1 sensitization
    
Clinical: rosacea flares → IL-1β ↑ → B1R expression ↑ → bradykinin (from KLK5) has more
    receptor to bind → worse neurogenic component during active flares vs. remission
```

## ACE-I Bradykinin Paradox in T1DM Rosacea

**ACE-I dual effect:**
```
ACE (angiotensin-converting enzyme) has two substrate activities:
    (1) Ang I → Ang II (primary; blocked by ACE-I → anti-NLRP3/NF-κB benefits; run_092)
    (2) Bradykinin → inactive fragments (kininase II activity; blocked by ACE-I → bradykinin ↑)
    
ACE-I (lisinopril, ramipril, enalapril):
    Ang II ↓ → NLRP3 Signal 2 ↓ + NF-κB ↓ [BENEFIT; run_092]
    Bradykinin accumulates → B2R → TRPV1 sensitization ↑ [PROBLEM: worsened neurogenic flushing]
    
Net effect in T1DM rosacea patients on ACE-I:
    Some benefit (RAAS anti-inflammatory) + some worsening (bradykinin B2R → TRPV1)
    → Clinical observation: subset of T1DM rosacea patients on ACE-I report worsened flushing
    → Mechanism was not previously identified in framework — now explained
    
Management implication:
    T1DM rosacea on ACE-I with worsened flushing: consider switch to ARB
    ARBs (losartan, irbesartan): block AT1R directly (Ang II cannot bind → NLRP3 ↓)
    But do NOT inhibit ACE → bradykinin is normally degraded → NO bradykinin accumulation
    ARBs: same NLRP3/NF-κB benefit as ACE-I WITHOUT bradykinin side effect
    
→ For T1DM rosacea specifically: ARBs may be preferred over ACE-I where flushing is
  severe or ACE-I cough/flush is a clinical problem. ARBs additionally:
  Ang II → AT2R (not blocked by ARBs) → bradykinin B2R activation via AT2R → actually
  slightly increases bradykinin action (AT2R → BK release; ARB-mediated AT2R activation)
  → This is mild and generally not clinically significant for skin flushing
```

**NEP (neprilysin) — alternative bradykinin degradation:**
```
NEP (neutral endopeptidase; CD10): cleaves bradykinin → inactive
    → NEP is the main non-ACE pathway for bradykinin clearance
    → Sacubitril (ARNI: sacubitril/valsartan) blocks NEP: → bradykinin further accumulates
    → Sacubitril/valsartan: NOT appropriate for T1DM rosacea patients with active flushing
    → Mentioned for completeness (mentioned in run_092 as not standard T1DM agent)
```

## Kill Criteria

**Kill A: KLK5 → Kininogen Cleavage Is Not Confirmed in Human Rosacea Dermis**
Yoon 2007 shows KLK5 kinin-generating activity in vitro. Eissa 2011 shows elevated kallidin/bradykinin in rosacea skin but does not directly demonstrate KLK5 as the kallikrein responsible (plasma kallikrein from KLKB1 is also possible).
**Status:** Partially concerning for specificity. KLK5 is the dominant tissue kallikrein in rosacea (overexpressed 5-10× in rosacea vs controls; multiple studies). KLK5 kinin-generating ability is established (Yoon 2007). The combined evidence that KLK5 is elevated in rosacea + KLK5 cleaves kininogens + kallidin elevated in rosacea skin makes the mechanistic chain strong even without direct KLK5-specific kinin generation in rosacea biopsies. Framework position: plausible, not definitively confirmed; clinically not actionable beyond existing KLK5 management.

**Kill B: Bradykinin B2R Sensitization of TRPV1 Is Not Rosacea-Specific — It's a General Nociception Mechanism**
The B2R → PKC → TRPV1 phosphorylation is a well-established nociception mechanism. There is no rosacea-specific B2R/TRPV1 data.
**Status:** Not killed. The TRPV1 overexpression and sensitization in rosacea (run_015) is established; B2R-mediated TRPV1 sensitization is the specific mechanism by which KLK5 (elevated in rosacea Loop 1) amplifies M8 neurogenic signaling. The mechanistic chain is complete (KLK5 ↑ → kallidin ↑ [Eissa 2011] → B2R → PKC → TRPV1 sensitized). Rosacea-specific data on the final B2R → TRPV1 step is absent but the upstream biology is confirmed in rosacea.

**Kill C: ACE-I Bradykinin Paradox Is a Minor Clinical Effect, Not Clinically Meaningful**
ACE-I-induced bradykinin accumulation → TRPV1 sensitization is subtle; most patients don't report worsened flushing.
**Status:** Partially correct. The paradox is clinically relevant in the SUBSET of T1DM rosacea patients who specifically report flushing worsening on ACE-I (a real clinical subgroup). For these patients, the ARB switch is now mechanistically justified. For patients without worsened flushing on ACE-I, the Ang II → NLRP3 benefits dominate. The paradox is a specific clinical decision tool, not a general counter-indication.
