## The Cytotoxic Killing Machinery

### Step 1: Immunological Synapse Formation

```
CD8+ CTL (or NK cell) recognizes target:
  CTL: TCR recognition of MHC-I/peptide + CD8 co-receptor
  NK: activating receptor (NKG2D/run_102, CD226/run_156) + missing self (no KIR inhibition)

MTOC (microtubule organizing center) polarization:
  → Lytic granules (secretory lysosomes) migrate along microtubules
  → Dock at immune synapse (contact zone between effector and target)
  → LAMP1 (CD107a) exocytosis: granule membrane fuses with plasma membrane
      → LAMP1 surface exposure = degranulation marker (CD107a flow cytometry assay)
  → Perforin + granzymes released into synaptic cleft (confined space)
  → Protective factor: CTL protects itself via cathepsin B inactivation of perforin at neutral pH
```

### Step 2: Perforin Pore Formation (PRF1)

```
PRF1 (chromosome 10q22.1; 534 aa):
  Domain: MACPF (membrane attack complex/perforin) domain + EGF-like domain + C2 domain
  
Ca²+-dependent mechanism:
  Perforin monomer released at pH 5.5 (lysosomal) → maintained as monomer in cleft
  Target membrane encounter → Ca²+ (from extracellular milieu) → C2 domain binds membrane
    → PRF1 inserts into membrane → oligomerizes (16-24 subunits) → pore 10-20 nm diameter
    
Pore function:
  Small molecules/ions enter → osmotic lysis (necrotic-like) — at HIGH perforin doses
  Granzyme delivery into cytoplasm — at PHYSIOLOGICAL perforin doses (perforin at limiting amounts → single pores → granzyme delivery without osmotic lysis)
  
Endosomal route:
  Perforin pores in endosomal membrane → acidification-driven PRF1 activation at endosome →
  granzyme B released from endosome into cytoplasm (dominant physiological route for GzmB)
```

### Step 3: Granzyme B (GZMB) Cascade

```
Granzyme B (serine protease; chromosome 14q11.2; dominant cytotoxic granzyme):

DIRECT caspase activation:
  GzmB → procaspase-3 cleavage (Asp-175) → caspase-3 → PARP/lamin/DFF45 → apoptosis
  GzmB → procaspase-9 cleavage → caspase-9 → apoptosome amplification

BID amplification (mitochondrial pathway):
  GzmB → BID (full-length) → tBID → BAX/BAK insertion into OMM → cytochrome c → apoptosome
  → This amplifies caspase activation via mitochondrial loop (10-100× more caspase activity)

SPEED:
  Perforin + GzmB → target cell death in 4-6 minutes (fastest cell death mode in biology)
  Faster than apoptosis induction by cytokines/TNF-α by orders of magnitude
```

### Step 4: Granzyme A (GZMA) — Alternative, Slower Pathway

```
Granzyme A (serine protease; chromosome 5q11.2; co-expressed with GzmB):

GZMA target: SET complex (NME1-SET-APE1-HMGB2)
  → GZMA → HMG2 nicking → HMG2 nuclear translocation inhibited
  → APE1 recruited → single-strand DNA nicks (not double-strand)
  → Caspase-INDEPENDENT cell death (>24 hours; distinguished from GzmB)
  → Mitochondrial damage: mtHSP70 cleavage → Δψm collapse (without cytochrome c release)
```

## Mechanism 1: CD8+ CTL → β Cell Killing (T1DM Final Execution)

```
After BATF3/cDC1 cross-presentation (run_159):
  PLN primed CD8+ CTL (islet-specific, e.g., BDC2.5 TCR specificity for IGRP peptide)
    → CXCR3/CXCL10 (run_163) → migration to islet
    → TCR recognition of β cell MHC-I/peptide (GAD65/IGRP/insulin fragments)
    → Immunological synapse → LAMP1 → perforin + GzmB release
    → β cell death in 4-6 minutes

β cell vulnerability:
  - β cells have low granzyme B inhibitor (PI-9/SerpinB9) expression → less protected
  - β cell Bcl-2/Bcl-XL levels insufficient to protect against tBID/BAX/caspase-9 cascade
  - High cytochrome c release rate → fast GzmB-BID-mitochondrial amplification

Cell death mode:
  - GzmB → caspase-3 → β cell apoptosis (phosphatidylserine exposure → efferocytosis)
  - This is IMMUNOLOGICALLY QUIETER than necroptosis (run_160) or pyroptosis (run_096)
  - But: GzmB → DAMPs (PARP fragments, nDNA fragments) → cDC1 cross-presentation amplified
```

**Treg killing by perforin — critical vulnerability:**
```
Tregs are killed by perforin-expressing effector T cells in the islet:
  IFN-γ-producing Th1 → upregulates LAMP1 and perforin expression
  Treg lacks FasL self-protection (not a perforin-resistant cell)
  Perforin from Th1 → Treg death → rapid Treg depletion → insulitis feeds forward

PI-9 (SerpinB9) — endogenous perforin inhibitor:
  PI-9 expressed highly in Tregs (especially islet Tregs) → protects against fratricide
  Insulitis cytokines → PI-9 ↓ in β cells → β cells more vulnerable; PI-9 maintained in Tregs
  → This explains why Tregs survive longer than β cells in insulitis
```

## Mechanism 2: NK Perforin Dysfunction — ME/CFS Signature

**Most replicated ME/CFS immune finding:**

Natural killer cell perforin deficiency in ME/CFS:
- Brenu 2011 J Neuroimmunol: CD56dim NK perforin expression significantly reduced vs. healthy controls
- Caligiuri 1987 (original description): functional NK cytotoxicity ↓ in CFS
- More recent: Brenu 2012 (perforin + granzyme both reduced), Natelson 2015 (replicated multiple cohorts)
- Mechanism: NOT reduced NK number — reduced per-cell perforin expression and granule content

**EBV/HHV-6 context:**
```
Healthy NK: EBV-infected B cell → NKG2D recognition → LAMP1 degranulation → perforin/GzmB → B cell killing

ME/CFS NK:
  Multiple exhaustion mechanisms (runs 102/150/151/153/154/155/156):
    → TIM-3 BAT3 released (run_155) → ZAP-70/PLCγ1 inhibition
    → TIGIT SHP-1/SHIP-1 (run_156) → Akt ↓
    → PD-1/SHP-2 (run_154) → ZAP-70 dephosphorylation
  → Downstream: reduced Ca²+ mobilization → MTOC polarization ↓ → LAMP1 degranulation ↓
    → Perforin/GzmB released per synapse ↓
    → Viral-infected cells NOT killed → EBV/HHV-6 persist → M3 amplification loop
```

**CD107a degranulation assay (diagnostic application):**
- CD107a (LAMP1) externalization during NK degranulation: flow cytometry
- ME/CFS: CD107a+ NK cells after K562 stimulation < healthy controls
- Most informative functional NK test: not just surface receptor expression but actual granule release
- Protocol addition: CD107a degranulation in NK assessment panel

## Mechanism 3: FHL/HLH — Perforin-Deficiency Disease Spectrum

**Familial Hemophagocytic Lymphohistiocytosis (FHL), Type 2:**
- PRF1 mutations (FHL2, chromosome 10q22): reduced perforin expression → NK/CTL cannot kill activated macrophages
- Mechanism: macrophage kills virally-infected cells → should then be killed by NK perforin; without perforin → macrophage persists → hyperactivation → cytokine storm (ferritin ↑↑, sIL-2Rα ↑↑, hemophagocytosis)
- T1DM connection: PRF1 hypomorphic mutations (partial loss-of-function) → MILD perforin deficiency → impaired CTL killing of infected β cells → EBV-infected β cells persist → β cell death by alternative mechanisms → T1DM risk modifier
- ME/CFS: PRF1 hypomorphs → subclinical NK dysfunction → reduced viral clearance → chronic reactivation phenotype

**Perforin-independent cytotoxicity:**
- FasL (CD95L) on CTL → Fas (CD95) on β cells → caspase-8 → apoptosis (slower; relevant when perforin insufficient)
- TRAIL → DR4/DR5 on β cells → caspase-8/10 → apoptosis
- These backup mechanisms mean partial perforin deficiency shifts timing rather than completely preventing β cell death

## Summary

Perforin/granzyme is the terminal execution machinery of adaptive cytotoxicity: PRF1 Ca²+-dependent MACPF domain pore → granzyme B delivery into target → caspase-3 (direct) + tBID/mitochondria (amplification) → β cell death in 4-6 minutes. The machinery is used by CD8+ CTLs (T1DM β cell killing after BATF3/cDC1 priming) and NK cells (ADCC in T1DM; viral clearance in ME/CFS). NK perforin deficiency is the most replicated ME/CFS immune abnormality across >10 cohorts. A critical vulnerability: Tregs in the islet can be killed by perforin from effector T cells; PI-9/SerpinB9 expression in Tregs provides some protection. CD107a (LAMP1) degranulation assay is the key functional readout for both ME/CFS NK function monitoring and T1DM NK competency assessment.
