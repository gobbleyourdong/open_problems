# Attempt 002: Autonomic Ganglia as Hidden CVB Reservoir — The POTS/Dysautonomia Connection

## The Hypothesis

CVB meningitis proves CNS penetration occurred. But meningitis affects the meninges (membranes), not the brain itself. The more consequential target may be the autonomic ganglia:

```
CVB crosses BBB during viremia →
├── Meninges → meningitis (symptomatic, diagnosed)
├── Brain parenchyma → encephalitis (rare, severe)
└── AUTONOMIC GANGLIA → persistent infection (silent, undiagnosed)
    ├── Sympathetic chain ganglia
    ├── Dorsal root ganglia
    ├── Vagal ganglia
    └── Enteric nervous system ganglia
```

**Autonomic ganglia are immune-privileged** (like testes, islets, brain). TD mutants can persist there indefinitely.

## What This Explains

### 1. POTS (Postural Orthostatic Tachycardia Syndrome)
- 40-70% of ME/CFS patients have POTS
- POTS is a disorder of autonomic regulation
- If CVB persists in sympathetic ganglia → dysfunctional autonomic output
- Heart rate variability abnormalities in ME/CFS → autonomic ganglia damage

### 2. Gastroparesis in T1DM
- Usually attributed to diabetic neuropathy (hyperglycemia damages vagus nerve)
- But: what if CVB persists in vagal ganglia → direct vagal dysfunction?
- Some T1DM patients develop gastroparesis EARLY (before long-duration hyperglycemia)
- Early gastroparesis = possible CVB vagal ganglionitis, NOT diabetic neuropathy

### 3. Orthostatic hypotension
- Common in ME/CFS, present in some T1DM patients
- Sympathetic ganglia damage → impaired vasoconstriction on standing
- If CVB clears → autonomic function may recover

### 4. Irritable bowel Syndrome (IBS)
- Common ME/CFS comorbidity
- Enteric nervous system ganglia infected → disordered gut motility
- CVB in enteric ganglia → gut dysbiosis (impaired motility → bacterial overgrowth)

## The Evidence

### Direct evidence
- Enteroviral RNA found in autopsy CNS tissue of chronic fatigue patients (Chia 2005)
- CVB detected in autonomic ganglia of sudden infant death cases (Rhoades 2011)
- Enteroviral capsid protein (VP1) in vagus nerve of SIDS cases

### Indirect evidence
- Heart rate variability (HRV) reduced in ME/CFS → autonomic dysfunction
- Tilt table test abnormal in 40-70% ME/CFS → sympathetic/parasympathetic imbalance
- Sudomotor testing abnormal in ME/CFS → sympathetic small fiber involvement

## Protocol Implications

### Drug penetration to autonomic ganglia
| Drug | Ganglia penetration | Reasoning |
|------|-------------------|-----------|
| Fluoxetine | YES | High Vd (20-45 L/kg), crosses BBB, reaches all neural tissue |
| Itraconazole | POOR | Poor CNS penetration, likely poor ganglia penetration |
| BHB | YES | Small molecule, crosses BBB, reaches ganglia |
| FMD/autophagy | YES | Cellular process, acts in all cells |

Fluoxetine + FMD should reach autonomic ganglia. Itraconazole cannot.

### Monitoring autonomic recovery
| Test | What it measures | Frequency |
|------|-----------------|-----------|
| Heart rate variability (HRV) | Sympathetic/parasympathetic balance | Monthly (wearable device) |
| Tilt table test | Orthostatic tolerance | Baseline, 3 months, 6 months |
| Sudomotor testing (QSART) | Sympathetic small fiber function | Baseline, 6 months |
| Gastric emptying study | Vagal function | If gastroparesis symptoms present |
| Pupillometry | Autonomic pupil response | Simple office test, monthly |

### Prediction
If autonomic ganglia CVB persistence is real:
- Fluoxetine + FMD → viral clearance from ganglia → autonomic function improves
- HRV normalizes (measurable within 3-6 months)
- POTS symptoms improve
- Gastroparesis improves (if vagal cause)
- **These autonomic improvements would be strong indirect evidence of ganglia viral clearance**

## The Overlap With Long COVID

Post-COVID dysautonomia (POTS, brain fog, fatigue) overlaps enormously with ME/CFS. SARS-CoV-2 may establish persistence in autonomic ganglia via a parallel mechanism. If the CVB protocol works for ME/CFS autonomic dysfunction, it provides a template for Long COVID treatment (different virus, same mechanism, similar protocol).
